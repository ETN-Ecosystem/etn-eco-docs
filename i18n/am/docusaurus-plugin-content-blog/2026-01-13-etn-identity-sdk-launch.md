**­Ъџђ рІГрЇІрІі рѕЇрЅђрЅх: ETN Identity SDK v1.0.0**

рІе **ETN Identity SDK** ріарѕЂріЋ рЅа **npm рѕІрІГ рЅарЅђрїЦрЅ│** рѕўрѕѕрЅђрЅЂріЋ рЅаріЕрѕФрЅх ріЦріЊрЅарѕхрѕФрѕѕріЋ! ріарІўрїІрїєрЅй рІѕрІГрѕЮ рІ░рЅбрІ│рЅбрІјрЅй (Developers) ріарѕЂріЋ рІ░рѕЁріЋріљрЅ▒ рІерЅ░рїарЅарЅђ рІе OIDC рѕЏрѕерїІрїѕрїФріЋ ріЦріЊ ріЦріЋріеріЋ рІерѕѕрѕй рІерЅ░рїарЅЃрѕџ рѕЮрІЮрїѕрЅБріЋ рЅа TON Blockchain рѕІрІГ рѕІрѕѕрІЇ рѕѕ ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рЅарѕЏрѕЮрїБрЅхрЇБ рЅарїЦрЅѓрЅх рІері«рІх рѕўрѕхрѕўрѕ«рЅй рЅЦрЅ╗ "Sign in with ETN" (рЅа ETN рІГрїЇрЅА) рІерѕџрѕѕрІЇріЋ рЅБрѕЁрѕф рІѕрІ░ React рѕўрЅ░рїЇрЅарѕфрІФрІјрЅ╗рЅИрІЇ рѕЏрІІрѕЃрІх рІГрЅйрѕІрѕЅрЇб

<!--truncate-->

**рѕѕріарЇІрїБріЮ рѕхрѕГрїГрЅх рІерЅ░рѕѕрЅђрЅђ**

## ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рЅа TON рѕІрІГ рІе Web3 рѕЏрѕерїІрїѕрїФріЋ рѕѕрѕЏрЅђрѕІрїарЇЇ рІГрЇІрІірІЇріЋ Identity SDK рїђрѕўрѕе

**ріарІ▓рѕх ріарЅарЅБрЇБ рібрЅхрІ«рїхрІФ Рђћ рїЃріЋрІІрѕф 13рЇБ 2026** Рђћ рІе ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рІЏрѕг рІе **ETN Identity SDK (v1.0.0)** ріЋ рІГрЇІрІі рѕЇрЅђрЅх ріарЅарѕ░рѕерЇб рІГрѕЁ ріЃрІГрѕѕріЏ ріарІ▓рѕх рІеріарІўрїІрїєрЅй (developer) рѕўрѕБрѕфрІФ рѕхрЅЦрѕхрЅЦрЇБ рІ░рѕЁріЋріљрЅ▒ рІерЅ░рїарЅарЅђ "Sign in with ETN" рЅ░рїЇрЅБрѕГріЋ рЅа TON Blockchain рѕІрІГ рЅБрѕЅ рІФрѕЇрЅ░рѕЏріерѕѕ рѕўрЅ░рїЇрЅарѕфрІФрІјрЅй (dApps) рІЇрѕхрїЦ рѕѕрѕЏрІІрѕЃрІх рЅ│рѕхрЅд рІерЅ░рІўрїІрїђ ріљрІЇрЇб

:::info рІерѕЏрІФрѕ╗рѕЏ рѕЏрѕхрЅ│рІѕрѕ╗
**ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ (ETN Ecosystem)** рЅа **TON Blockchain** рѕІрІГ рІерЅ░рїѕріљрЅБ ріЦріЊ рЅаріарЇЇрѕфріФ рІЇрѕхрїЦ рЅарІ▓рїѓрЅ│рѕЇ рѕЏріЋріљрЅх ріЦріЊ рЅарЇІрІГріЊріЋрѕх ріаріФрЅ│рЅйріљрЅх рѕІрІГ рІФрЅ░рі«рѕе рѕФрѕ▒ріЋ рІерЅ╗рѕѕ рЇЋрѕ«рїђріГрЅх ріљрІЇрЇб ріЦріЏ ріе Electroneum (ETN) рІѕрІГрѕЮ ріе Ethereum (ETH) рїІрѕГ рїЇріЋріЎріљрЅх **рІерѕѕріЋрѕЮ**::
:::

рІе ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рІ▓рїѓрЅ│рѕЇ ріарїѕрѕЇрїЇрѕјрЅХрЅ╣ріЋ рЅаріарЇЇрѕфріФ ріЦріЊ ріерІџрІФрѕЮ рЅарѕІрІГ ріЦрІФрѕ░рЇІ рѕ▓рѕёрІхрЇБ рІ░рѕЁріЋріљрЅ▒ рІерЅ░рїарЅарЅђ ріЦріЊ рѕѕрЅ░рїарЅЃрѕџ рѕЮрЅ╣ рІерѕєріљ рІерѕЏріЋріљрЅх ріарѕхрЅ░рІ│рІ░рѕГ рІѕрѕ│ріЮ рЇЇрѕІрїјрЅх рѕєріЌрѕЇрЇб ріарІ▓рѕ▒ SDK рЅарЅ░рІѕрѕ│рѕ░рЅА рІерЅЦрѕјріГрЅ╝рІГріЋ рѕЮрѕхрїарѕФрІјрЅй (cryptography) ріЦріЊ рЅарѕўрІ░рЅаріЏ рІерІхрѕГ рѕЏрѕерїІрїѕрїФрІјрЅй рѕўріФріерѕЇ ріЦріЋрІ░ рІхрѕЇрІхрІГ рѕєріќ рІФрїѕрѕѕрїЇрѕІрѕЇрЇБ рІГрѕЁрѕЮ ріарІўрїІрїєрЅй рїаріЋріФрѕФ рІе OpenID Connect (OIDC) рЇЇрѕ░рЅХрЅйріЋ рЅарѕ│рѕЮріЋрЅ│рЅх рѕ│рІГрѕєріЋ рЅарІ░рЅѓрЅЃрІјрЅй рІЇрѕхрїЦ ріЦріЋрІ▓рЅ░рїѕрЅЦрѕЕ рІФрѕхрЅйрѕІрЅИрІІрѕЇрЇб

### рІе Web3 рѕЏріЋріљрЅхріЋ рѕЏрЅЃрѕѕрѕЇ

рІе ETN Identity SDK рІе OAuth 2.0 рІерїЇрІ┤рЅ│ рі«рІх рЇЇрѕ░рЅхрЇБ рІерЅХрігріЋ рѕйріГрѕГріГрѕфрЅх (rotation)рЇБ ріЦріЊ рІеріГрЇЇрѕѕ-рїірІю (session) ріарѕхрЅ░рІ│рІ░рѕГ рІЇрѕхрЅЦрѕхрЅЦріљрЅхріЋ рІФрѕхрІѕрїЇрІ│рѕЇрЇб ріарѕхрЅђрІхрѕъ рІерЅ░рїѕріљрЅБрЇБ рѕЎрѕЅ рѕѕрѕЎрѕЅ рЅа type-safe рІерЅ░рЇЃрЇѕ рІе React рѕІрІГрЅЦрѕерѕф рЅарѕЏрЅЁрѕерЅЦрЇБ рІе ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рЅђрїБрІЕріЋ рІерЇІрІГріЊріЋрѕх ріЦріЊ рІерѕЏрѕЁрЅарѕФрІі рѕўрЅ░рїЇрЅарѕфрІФрІјрЅй рЅхрІЇрѕЇрІх рѕѕрѕџрїѕріљрЅА ріарІўрїІрїєрЅй рІерЅ┤ріГріњріГ ріЦріЋрЅЁрЇІрЅХрЅйріЋ ріЦрІерЅђріљрѕ░ ріљрІЇрЇб

"рІ░рѕЁріЋріљрЅ▒ рІерЅ░рїарЅарЅђ рѕЏрѕерїІрїѕрїФріЋ ріерЅБрІХ рѕўрїѕріЋрЅБрЅх рѕѕріарІ│рІ▓рѕх ріарІўрїІрїєрЅй рЅхрѕЇрЅЂ рЇѕрЅ░ріЊ ріљрІЇрЇБ" рІГрѕІрѕЅ рІе ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рѕўрѕхрѕФрЅй **Jason "JP" Peters**рЇб "рЅарІџрѕЁ SDK ріарѕЏріФріЮріљрЅхрЇБ рЅаріЦріЏ ріарІЇрЅ│рѕе рѕўрѕерЅЦ рІЇрѕхрїЦ рѕЏріЋріљрЅх ріЦріЋрІ┤рЅх ріЦріЋрІ░рѕџрѕ░рѕФ рѕўрІ░рЅаріЏ ріЦрІФрІ░рѕерїЇріЋ ріљрІЇрЇб рІе DeFi рѕўрІхрѕеріГрѕЮ рѕєріљ рІерѕЏрѕЁрЅарѕФрІі dApp ріЦрІерїѕріљрЅА рЅбрѕєріЋрЇБ ріЦріЏ ріерЅарѕхрЅ░рїђрѕГрЅБ рІерІ░рѕЁріЋріљрЅх ріЦріЊ рІеріГрЇЇрѕѕ-рїірІю рїЦрїѕріЊріЋ ріерЅБрІх рѕхрѕФ ріЦрІерѕ░рѕФріЋрЇБ ріЦріЊріЋрЅ░ рЅаріаріЋрІх рІері«рІх рѕўрѕхрѕўрѕГ рЅЦрЅ╗ рЅ░рїарЅЃрѕџрІјрЅйріЋ рІ░рѕЁріЋріљрЅ▒ рЅарЅ░рїарЅарЅђ рѕЂріћрЅ│ рѕЏрѕхрѕўрІЮрїѕрЅЦ рЅхрЅйрѕІрѕІрЅйрѕЂрЇб"

### рІе SDK рЅЂрѕЇрЇЇ рЅБрѕЁрѕфрІјрЅй

*   **рЇѕрїБріЋ рІЇрѕЁрІ░рЅх:** рѕѕ React рѕўрЅ░рїЇрЅарѕфрІФрІјрЅй plug-and-play `<ETNConnectButton />` ріаріФрѕЇріЋ рІФріФрЅхрЅ│рѕЇрЇб
*   **рІерібріЋрЅ░рѕГрЇЋрѕФрІГрІЮ-рІ░рѕерїЃ рІ░рѕЁріЋріљрЅх:** рѕѕ OIDC Authorization Code Flow ріе PKCE ріЦріЊ ріеріарІЇрЅХрѕЏрЅ▓ріГ Token Rotation рїІрѕГ рѕЎрѕЅ рІхрїІрЇЇ рІГрѕ░рїБрѕЇрЇб
*   **Type-Safe:** рІері«рІх ріарѕхрЅ░рѕЏрѕЏріЮріљрЅхріЋ ріЦріЊ рІеріарІўрїІрїЁ рѕЮрѕГрЅ│рѕЏріљрЅхріЋ рѕѕрѕЏрѕерїІрїѕрїЦ рѕЎрѕЅ рЅарѕЎрѕЅ рЅа TypeScript рЅ░рїйрЇЈрѕЇрЇб
*   **ріГрЇЇрЅх рѕЮріЋрїГ:** рѕЎрѕЅ рЅарѕЎрѕЅ рїЇрѕЇрїй ріЦріЊ рѕѕрѕЏрѕЁрЅарѕерѕ░рЅЦ ріарѕхрЅ░рІІрЇЁрІќ рЅа GitHub рѕІрІГ рІГрїѕріЏрѕЇрЇб

### ріЦріЋрІ┤рЅх ріЦріЋрІ░рѕџрїђрѕўрѕГ

SDK рЅарІГрЇІрІірІЇ рІе npm рѕўрІЮрїѕрЅЦ (registry) рЅаріЕрѕЇ рІѕрІ▓рІФрІЇріЉ рІГрїѕріЏрѕЇрЇб ріљрїѕрѕГ рїЇріЋрЇБ рѕІрІГрЅЦрѕерѕфрІЇріЋ ріерѕЏрІІрѕЃрІх рЅарЇірЅхрЇБ ріарІўрїІрїєрЅй ріарѕхрЇѕрѕІрїірІЇріЋ рІе API рѕЮрѕхріГрѕГріљрЅХрЅй (Client ID ріЦріЊ Client Secret) рѕѕрѕЏрїЇріўрЅх рѕўрЅ░рїЇрЅарѕфрІФрЅИрІЇріЋ рѕЏрѕхрѕўрІЮрїѕрЅЦ ріарѕѕрЅБрЅИрІЇрЇб

1.  **рѕўрЅ░рїЇрЅарѕфрІФрІјріЋ рІФрѕхрѕўрІЮрїЇрЅА:** рІеріарІўрїІрїЁ рѕЮрѕхріГрѕГріљрЅХрЅйрІјріЋ рѕѕрѕЏрѕўріЋрїерЅх рІе **[ETN Account Platform](https://account.etnecosystem.org/)** ріЋ рІГрїјрЅЦріЎрЇб
2.  **SDK ріЋ рІГрїФріЉ:** рЅарЇЋрѕ«рїђріГрЅхрІј рІЇрѕхрїЦ рІГрѕЁріЋріЋ рЅхрІЋрІЏрІЮ рІФрѕѓрІ▒рЇА-
    ```bash
    npm install @etn-ecosystem/identity-sdk
    ```
3.  **рІГрїѕріЋрЅА:** рІерѕўрїђрѕўрѕфрІФрІЇріЋ рІерѕўрїЇрЅбрІФ (sign-in) рЇЇрѕ░рЅхрІјріЋ рѕѕрѕўрЅ░рїЇрЅарѕГ рЅа [GitHub](https://github.com/ETN-Ecosystem/ETN-Identity-SDK) рѕІрІГ рІФрѕѕрІЇріЋ рѕ░ріљрІх рІГріерѕЇрѕ▒рЇб

### рѕхрѕѕ ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ

рІе ETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рЅа TON Blockchain рѕІрІГ рІерЅ░рїѕріљрЅБ рІФрѕЇрЅ░рѕЏріерѕѕ ріарІЇрЅ│рѕе рѕўрѕерЅЦ рѕ▓рѕєріЋрЇБ рЅарЇІрІГріЊріЋрѕх ріаріФрЅ│рЅйріљрЅх ріЦріЊ рЅарѕЅрІЊрѕІрІі рѕЏріЋріљрЅх (sovereign identity) рЅ░рїарЅЃрѕџрІјрЅйріЋ рІерѕџрІФрЅарЅЂ рІ▓рїѓрЅ│рѕЇ рѕўрЇЇрЅхрѕёрІјрЅйріЋ рѕѕрѕўрЇЇрїарѕГ рІерЅ░рїарѕѕрѕѕ ріљрІЇрЇб ріе ETN Vibe рѕўрІхрѕеріГ ріЦрѕхріе ріарІ▓рѕ▒ Identity SDK рІхрѕерѕхрЇБ ETN рѕѕрЅ░рїѕріЊріўрЅй рІ▓рїѓрЅ│рѕЇ ріарЇЇрѕфріФ рѕўрѕ░рѕерЅ░ рѕЇрѕЏрЅхріЋ ріЦрІерїѕріљрЅБ ріљрІЇрЇб
