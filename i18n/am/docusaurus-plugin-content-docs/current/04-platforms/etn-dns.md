# ETN DNS

## ያልተማከለ የጎራ ስም አገልግሎት – ምርምር

ETN DNS በቶን ብሎክቼይን ላይ የተገነባ ያልተማከለ የጎራ ስም ስርዓት ሲሆን፣ ተጠቃሚዎች ልዩ የሆኑ ንዑስ ጎራዎችን በ`.etnetsacoin.ton` ስር እንደ NFTs እንዲጫረቱ እና እንዲፈጥሩ ያስችላል፣ ይህም እውነተኛ ዲጂታል ባለቤትነትን ይሰጣል። መድረኩ የጨረታ ስርዓት እና "አሁን ይግዙ" አማራጭ ያለው ሲሆን፣ ሁሉም ግብይቶች በETN ቶከኖች ብቻ የሚከናወኑ ሲሆን፣ ለእነዚህ ዲጂታል ማንነቶች ተለዋዋጭ ገበያ ይፈጥራል።
