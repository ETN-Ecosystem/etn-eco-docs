# рІеETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рѕўрІхрѕері«рЅй

рІГрѕЁ ріГрЇЇрѕЇ рЅаETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рІЇрѕхрїЦ рІФрѕЅрЅхріЋ рІерЅ░рѕѕрІФрІЕ рѕўрІхрѕері«рЅй ріЦріЊ ріарїѕрѕЇрїЇрѕјрЅХрЅй ріарїарЅЃрѕІрІГ ріЦрІГрЅ│ рІГрѕ░рїБрѕЇрЇб

­ЪїЇ **рІеETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рѕўрІхрѕері«рЅй ріарїарЅЃрѕІрІГ ріЦрІГрЅ│**

**рІеETN рѕхріљ-рѕЮрѕЁрІ│рѕГ** рЅа**рЅХріЋ рЅЦрѕјріГрЅ╝рІГріЋ** рѕІрІГ рІерЅ░рїѕріљрЅБ ріЦріЊ рЅа**$ETN рЅХріеріЋ** рІерЅ░рїјрѕІрЅарЅ░ ріЦрІФрІ░рїѕ рІФрѕѕ рІФрѕЇрЅ░рѕЏріерѕѕ рѕўрІхрѕері«рЅй рѕхрЅЦрѕхрЅЦ ріљрІЇрЇб ріЦрІФріЋрІ│ріЋрІ▒ рѕўрІхрѕеріГ рІерЅ░рѕѕрІе рЇЇрѕІрїјрЅхріЋРђћ**рІерЇІрІГріЊріЋрѕх рѕЏріФрЅ░рЅхріЋ**рЇБ **рЅхрѕЮрѕЁрѕГрЅхріЋ** ріЦріЊ **рІ▓рїѓрЅ│рѕЇ рЅхрѕФріЋрѕхрЇјрѕГрѕюрѕйріЋріЋ**РђћрЅаріарЇЇрѕфріФ рѕІрІГ рѕЇрІЕ рЅхріЕрѕерЅх рЅарѕўрѕхрїарЅх рІГрЇѕрЅ│рѕЇрЇб

---

## ­ЪДЕ рІерѕўрІхрѕеріГ рѕЏрІЇрїФ

ріерІџрѕЁ рЅарЅ│рЅй рЅаETN рѕхріљ-рѕЮрѕЁрІ│рѕГ рІЇрѕхрїЦ рІФрѕЅ **27 рѕўрІхрѕері«рЅй** ріарїарЅЃрѕІрІГ рІЮрѕГрІЮрѕГ рЅђрѕГрЅДрѕЇ:

| ­Ъџђ **рІерѕўрІхрѕеріГ рѕхрѕЮ** | ­ЪЈи№ИЈ **рѕЂріћрЅ│** | ­ЪЊІ **рѕўрїЇрѕѕрїФ** |
|----------------------|---------------|---------------------|
| ­Ъїљ **ETN DNS** | ­ЪДф `рЅарѕЇрѕЏрЅх рѕІрІГ` | рІФрѕЇрЅ░рѕЏріерѕѕ рІерїјрѕФ рѕхрѕЮ рѕхрѕГрІЊрЅхрЇб ріЋрІЉрѕх рїјрѕФрІјрЅй ріЦріЋрІ░ NFTs рЅа`.etnetsacoin.ton` рѕхрѕГрЇб |
| ­ЪќЦ№ИЈ **ETN HOSTING** | ­Ъћљ `рІЮрїЇ рЅцрЅ│` | рІеWeb2/3 рїБрЅбрІФрІјрЅйріЋ рЅарЅХріЋ рѕЏріерѕЏрЅ╗ рЅарѕўрїарЅђрѕЮ рІФрѕхрЅ░ріЊрїЇрІ│рѕЇрЇб рІерЅ░рїѕрѕІрЅбрїдрѕй рЇЋрѕ«ріГрѕ▓ рІФрЅђрѕГрЅБрѕЇрЇб рЅа$ETN рІГріерЇѕрѕІрѕЇрЇб |
| ­ЪЊџ **ETN LEARN** | РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` | рѕѕрЅхрѕЮрѕЁрѕГрЅх рѕ░рїфрІјрЅй ріЦріЊ рЅ░рѕЏрѕфрІјрЅй рІеWeb3 рІерѕўрѕЏрѕфрІФ рѕўрІхрѕеріГрЇб рѕЂрѕЅрѕЮ ріГрЇЇрІФрІјрЅй рЅа$ETNрЇб |
| ­Ъњ│ **ETN PAY** | РЈ│ `рІІріЊрІЇ рѕўрІхрѕеріГ рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рѕўрїЇрЅбрІФ + WooCommerce рЅ░рѕ░ріф рЅарЅцрЅ│рЇб POS + ERPріЋ рІГрІ░рїЇрЇІрѕЇрЇб |
| ­ЪЏњ **ETN рїѕрЅарІФ** | ­ЪћД `рЅарѕўрїѕріЋрЅБрЅх рѕІрІГ` | рѕѕрІЋрЅЃрІјрЅй ріЦріЊ ріарїѕрѕЇрїЇрѕјрЅХрЅй рІФрѕЇрЅ░рѕЏріерѕѕ рІеріб-рі«рѕюрѕГрѕх рѕўрІхрѕеріГрЇб |
| ­ЪЊќ **рѕўрѕ░рІѕрѕГ рѕўрїйрѕљрЇЇрЅх** | ­Ъџђ `рѕѕрѕўрѕЇрЅђрЅЁ рЅЁрѕГрЅЦ` | рЅ┤рѕїрїЇрѕФрѕЮ-рЅ░рІѕрѕІрїЁ ETN-рІерЅ░рїјрѕІрЅарЅ░ ріб-рѕўрїйрѕљрЇЇ рѕўрІ░рЅЦрѕГрЇб |
| ­ЪЊб **ETN ADs** | ­Ъћљ `рІЮрїЇ ріарѕЇрЇІ` | рѕѕрЅ┤рѕїрїЇрѕФрѕЮ ріЦріЊ рѕѕрІхрѕГ рІФрѕЇрЅ░рѕЏріерѕѕ рІерѕЏрѕхрЅ│рІѕрЅѓрІФ рїѕрЅарІФрЇб рЅа$ETN рІГріерЇѕрѕІрѕЇрЇб |
| ­ЪЈЏ№ИЈ **ETN Council** | РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` | рІерѕхріљ-рѕЮрѕЁрІ│рѕГ рЅЂрїЦрїЦрѕГ ріарѕхрЅ░рІ│рІ░рѕГ DAOрЇб рѕўрїЇрЅбрІФ рЅаNFTрЇБ рЅарЅХріеріЋ рЅЦрІЏрЅх ріарІГрІ░рѕѕрѕЮрЇб |
| ­Ъњ▒ **ETN-Ex** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рЅБрѕѕрЅЦрІЎ рѕЇрІЇрІЇрїЦ рЇѕрѕ│рѕйріљрЅх рІФрѕѕрІЇ DEX ріарїЇрѕфрїїрЅ░рѕГрЇб |
| ­Ъњ░ **ETN STAKING (jVault)** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рѕѕрѕйрѕЇрѕЏрЅх ETN рѕхрЅ┤ріфріЋрїЇ рІФрІхрѕГрїЅрЇб ріерЅХріЋ PoS рїІрѕГ рІерЅ░рїБрїБрѕўрЇб |
| ­ЪЉЦ **ETN рїђрѕўрІЊ** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рїЇрѕІрІіріљрЅх-рѕўрїђрѕўрѕфрІФ рІеріаріФрЅБрЅб рѕЏрѕЁрЅарѕФрІі рѕџрІ▓рІФ ріарІЇрЅ│рѕе рѕўрѕерЅЦрЇб |
| ­ЪЉеРђЇ­Ъњ╗ **ETN ріљрЇЃ-рѕІріЋрѕ░рѕГ** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рѕхрѕЏрѕГрЅх рі«ріЋрЅхрѕФрЅх ріГрЇЇрІФрІјрЅй рІФрѕѕрІЇ рІерЇЇрѕфрѕІріЋрѕх рІерѕхрѕФ рѕўрІхрѕеріГрЇб |
| ­ЪїЇ **ETN рїЅрІъ** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рѕѕрібрЅхрІ«рїхрІФ рІеWeb3 рІерїЅрІъ рІЮрѕГрІЮрѕГ ріЦріЊ рІерЅдрЅ│ рѕЏрѕхрІФрІБ рѕўрІхрѕеріГрЇб |
| ­Ъњ│ **ETN CARDS** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рЅаETN рѕірїѕрІЎ рІерѕџрЅйрѕЅ рѕЮріЊрЅБрІі ріФрѕГрІХрЅйрЇБ рЅаUSDT рѕірїФріЉ рІерѕџрЅйрѕЅрЇб |
| ­Ъћљ **ETN ріФрІЮріЊ** | РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` | рѕѕETN ріаріФрѕІрІі рЅђрІЮрЅЃрІЏ рЅдрѕГрѕ│рЇБ рЅаріарїѕрѕГ рІЇрѕхрїЦ рІерЅ░рѕ░рѕФрЇб |
| ­Ъје **ETN Templates** | ­ЪћД `рЅарѕўрїѕріЋрЅБрЅх рѕІрІГ` | рѕѕрЇѕрїБрѕфрІјрЅй рІеріЋрЅЦрѕерЅх рЅцрЅ░-рѕўрїйрѕљрЇЇрЅхРђћрЅаETN рІГріерЇѕрѕІрѕЇрЇб |
| ­ЪЊ╗ **ETN-FM** | ­ЪДф `рІерѕЎріерѕФ рѕхрѕГрїГрЅх` | рІерѕўрѕхрѕўрѕГ рѕІрІГ рѕгрІ▓рІ« рЅарѕЎріерѕФ рѕІрІГрЇб ETN-рЅ░рі«рѕГ рЅБрѕЁрѕфрІФрЅх рЅарЅЁрѕГрЅА рІГрѕўрїБрѕЅрЇб |
| ­Ъћб **ETN Numbers** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рЅаNFT рѕІрІГ рІерЅ░рѕўрѕ░рѕерЅ░ рІ▓рїѓрЅ│рѕЇ рѕЏріЋріљрЅх (рѕхрѕЮ-ріарѕЇрЅБ/2FA)рЇб ріеSSO рїІрѕГ рІГрїѕріЊріЏрѕЇрЇб |
| ­ЪдИРђЇРЎѓ№ИЈ **рІерібрЅхрІ«рїхрІФ рїарЅБрЅѓрІјрЅй** | РюЁ `рЅ░рїђрѕЮрѕ»рѕЇ` | ріЦрІЇріљрЅ░ріЏ рІерЅХріЋ рѕйрѕЇрѕЏрЅХрЅй рІФрѕІрЅИрІЇ NFT ріарѕЮрѕ│рІФрІјрЅйрЇб |
| ­ЪЏА№ИЈ **ETN Auth** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рЅаETN Numbers рІерЅ░рїјрѕІрЅарЅ░ рІеWeb3 SSO рѕхрѕГрІЊрЅхрЇб |
| ­Ъќ╝№ИЈ **ETN рїЅрѕірЅх** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | ETNріЋ рЅарѕўрїарЅђрѕЮ рЅ░рІѕрѕІрїЁ NFT рІерїѕрЅарІФ рЅдрЅ│рЇб |
| ­Ъњ╝ **ETN ріЦрЅЂрЅЦ** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рЅарЅЦрѕјріГрЅ╝рІГріЋ рІерЅ░рїјрѕІрЅарЅ░ рІерібрЅхрІ«рїхрІФ рЅЂрїарЅБ рЅАрІхріќрЅйрЇб |
| ­ЪЉеРђЇ­Ъњ╗ **ETN Devs** | РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` | рЅаETN рІЇрѕхрїЦ рІеріцрѕхріГрѕ«рІЇ рѕЮрІЋрѕФрЇјрЅй рІФрѕѕрІЇ рІерїѕріЋрЅб рЅЁрїЦрѕГ рѕўрІхрѕеріГрЇб |
| ­ЪЌъ№ИЈ **ETN Daily** | РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` | рЅХріЋ + рЅЦрѕјріГрЅ╝рІГріЋ рЅЦрѕјрїЇ ріЦріЊ рІеETN рІюріЊ рЇќрѕГрЅ│рѕЇ ріљрІЇрЇб |
| ­ЪћЌ **ETN Bio** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рЅарЅ▓рЇњріЋрїЇ + рІерЅдрѕГрѕ│ рІхрїІрЇЇ рІФрѕѕрІЇ рІеWeb3 рѕўрїѕрѕѕрїФ рїѕрїйрЇб |
| ­Ъцќ **NetsaAI** | РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` | рѕѕріЋрїЇрІХрЅй рІерЅ┤рѕїрїЇрѕФрѕЮ рЅдрЅх рїѕріЋрЅбрЇб ріГрЇЇрІФрІјрЅй рЅаETNрЇб |
| ­Ъћљ **ETN Authenticator** | ­ЪЋЊ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` | рѕѕрѕхріљ-рѕЮрѕЁрІ│рѕЕ OTP + 2FA рѕўрѕ│рѕфрІФрІјрЅйріЋ рІерѕџрІФрЅђрѕГрЅЦ TWAрЇб |

­ЪЊї **ріарЇѕ рЅ│рѕфріГ**:
- РюЁ `ріЦрІерѕ░рѕФ ріљрІЇ` РђЊ рЅђрїЦрЅ│ ріЦріЊ ріЋрЅЂ
- ­Ъћљ `рІЮрїЇ ріарѕЇрЇІ/рЅцрЅ│` РђЊ ріерЅ░рІѕрѕ░ріЉ рЅ░рїарЅЃрѕџрІјрЅй рїІрѕГ рѕўрѕъріерѕГ
- ­ЪћД `рЅарѕўрїѕріЋрЅБрЅх рѕІрІГ` РђЊ рЅаріЋрЅЃрЅх ріЦрІерЅ░рїѕріљрЅБ ріљрІЇ
- ­ЪДф `рЅарѕЇрѕЏрЅх рѕІрІГ` РђЊ рїйріЋрѕ░-рѕђрѕ│рЅБрІі рІ░рѕерїЃ рІѕрІГрѕЮ рІерѕўрїђрѕўрѕфрІФ рїЇріЋрЅБрЅ│
- РЈ│ `рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ` РђЊ рЅ│рЅЁрІХ рїЇріЋрЅБрЅ│ріЋ рЅарѕўрїарЅБрЅарЅЁ рѕІрІГ
