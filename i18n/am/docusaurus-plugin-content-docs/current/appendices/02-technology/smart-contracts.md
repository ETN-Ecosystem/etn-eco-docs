# ስማርት ኮንትራቶች

ስማርት ኮንትራቶች በቶን ብሎክቼይን ላይ የሚሰሩ፣ የስምምነቱ ውሎች በቀጥታ ወደ ኮድ የተጻፉ፣ በራሳቸው የሚፈጸሙ ኮንትራቶች ናቸው።

## አጠቃላይ እይታ

### ስማርት ኮንትራቶች ምንድን ናቸው?
- በራሳቸው የሚፈጸሙ ኮዶች
- በራስ-ሰር የሚተገበሩ
- ግልጽ አፈጻጸም
- የማይለወጡ ህጎች

### ጥቅሞች
- እምነት የለሽ አፈጻጸም
- በራስ-ሰር የሚሰሩ ሂደቶች
- የተቀነሰ ወጪዎች
- የተሻሻለ ቅልጥፍና
