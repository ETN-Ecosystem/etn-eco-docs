Sadaasa 17, 2024 Ekosistimii ETN tiif guyyaa seena-qabeessa ture, sababiin isaa eebbifamtoota marsaa jalqabaa **Ultimate Netsa Summer Bootcamp**, kan boinuun "**Ahadu Batch**" (ykn "Genesis Block") jennee moggaafne kabajneerra! Taateen kun **jalqabbii lallaafaa (soft launch) guutuu Ekosistimii ETN** ttiis tajaajileera, kunis waggoota shan keessatti Itoophiyaanota miiliyoona 30–50 gara blockchain TON tti fiduuf karoorra keenya guddaaf haala mijeesseera.

<!--truncate-->

Leenjiin ji'a lamaa cimaan kun, kan ETN Learn karaa kaffaltii malee dhihaate, barattoota kutaa 12ffaa Itoophiyaa 7,000 ol ta'aniif beekumsa murteessaa Daldala Forex, Cryptocurrency fi Teeknooloojii Blockchain, fi Seensa Web3 fi The Open Network (TON) milkaa'inaan hidhachiiseera.

### Ahadu Badge SBT Beeksisuu

Dumni sirna eebbaa sanaa beeksifamuu **Ahadu Badge** ti, kunis **Soulbound Token (SBT)** yaadannoo eebbifamtoota qara-bahiilota kanaaf kennamuudha. Ahadu Badge walumatti 192 blockchain TON irratti 'mint' kan godhaman yoo ta'u, tokkoon tokkoon isaanii mallattoo addaa Ahadu Batch tiin kan faayamanidha. SBTs dabarsuun hin danda'amne kunniin mallattoo milkaa'inaa qofa miti; **koorsiiwwan waltajjii ETN Learn hundaaf gahiinsa umrii guutuu** kan qabiyyee kaffaltii qabaniis dabalatu ni kennu. Kaka'umsi kun dhiheessii gatii fi hirmaannaa hawaasaaf adeemsa keenya inni olaanaa fakkeenya kan ta'u yoo ta'u, amanamummaa guddisee faayidaa qabatamaa teeknooloojii SBT ni agarsiisa.

Taateen kun tumsa Exness Limited, daldalaa (broker) addunyaa, kan Hoogan Regional Sales Baha Afrikaa sirna irratti argamuun caalaatti cimeera. Barattoonni mojuulii daldalaa (trading module) irratti qabxii olaanaa fidan badhaasawwan kan akka laaptoppii, taableetii, fi ismaartifoonii badhaafamaniiru.

Milkaa'inni waloo kun—eebbi milkaa'aa marsaa jalqabaa namoota Web3 baratan, fayyadama kalaqaa SBT ragaa fi faayidaaf oolchuu, fi jalqabbii lallaafaa ekosistimii keenyaa—seenaa guddinaa, humneessuu, fi hojiirra oolmaa addunyaa dhugaa hatattamaa ta'e uuma. Nuti humna namaa baratee fi hirmaatu, kan sochi, kalaqa, fi guddina waliigalaa Ekosistimii ETN tiif gumaachu, kallattiin ijaaraa jirra.
