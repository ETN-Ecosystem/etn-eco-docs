## Ekosistimiin ETN Mirkaneessa Web3 TON irratti Salphaa Taasisuuf Identity SDK Ofiseelaa Jalqabsiise

**ADDIIS ABABAA, Itoophiyaa â€” Amajjii 13, 2026** â€” Ekosistimiin ETN har'a gadi-lakkifama ofiseelaa **ETN Identity SDK (v1.0.0)** beeksiseera, kunis meeshaa misoomsitootaa haaraa fi humna qabeessa ta'ee fi tajaajila "Sign in with ETN" nageenyummaan isaa eegame appilikeeshiniiwwan gadi-lakkifamoo (dApps) TON Blockchain irra jiran keessatti walitti-fiduuf kan dizaayinii ta'eedha.

:::info Hubachiisa Adda Baasuu
**Ekosistimiin ETN (ETN Ecosystem)** pirojektii of-danda'aa **TON Blockchain** irratti ijaaramee fi eenyummaa dijitaalaa fi hirmaachisummaa faayinaansii Afrikaa keessatti xiyyeeffatedha. Nuti Electroneum (ETN) yookiin Ethereum (ETH) waliin hidhata **hin qabnu**.
:::

Ekosistimiin ETN tajaajiloota dijitaalaa isaa Afrikaa fi isaa olitti babal'isuu yeroo itti fufu kanatti, bulchiinsi eenyummaa nageenyummaan isaa eegamee fi fayyadamaaf mijaawaa ta'e fedhii murteessaa ta'eera. SDK'n haaraan kun akka riqicha gidduu 'cryptography' blockchain walxaxaa fi mirkaneessa weebii idilee ta'ee tajaajila, kunis misoomsitoonni adeemsa OpenID Connect (OIDC) cimaa ta'e torbanootaan osoo hin taane daqiiqaawwan keessatti akka hojiirra oolchan isaan dandeessisa.

### Eenyummaa Web3 Salphaa Taasisuu

ETN Identity SDK walxaxitummaa adeemsa koodii hayyamaa (authorization code flow) OAuth 2.0, nanaanna'iinsa token (token rotation), fi bulchiinsa seeshinii (session management) ni hambisa. Layibiraarii React duraan-ijaarame fi guutummaatti 'type-safe' ta'e dhiheessuudhaan, Ekosistimiin ETN gufuuwwan teeknikaa misoomsitoota dhaloota appilikeeshiniiwwan faayinaansii fi hawaasummaa itti aanu ijaaraniif jiru gadi-buusaa jira.

"Mirkaneessa nageenyummaan isaa eegame eegaluu irraa ijaaruun qabxiiwwan dadhabsiisoo misoomsitoota haaraatiif isaan guddaadha," jedhan **Jeesan "JP" Piitars**, Hundeessaan Ekosistimii ETN. "SDK kanaan, nuti eenyummaan netwoorkii keenya keessatti akkamitti akka hojjetu idileessaa (standardizing) jirra. Waltajjii DeFi yookiin dApp hawaasummaa ijaaraa yoo jiraattanis, nuti duuba-deebiin (in the background) hojii ulfaataa nageenyummaa fi eeggarsa seeshinii osoo hojjennuu, isin sarara koodii tokko qofaan fayyadamtoota nageenyummaan simachuu dandeessu."

### Amaloota Ijoo SDK

*   **Walitti-hidhiinsa Hatattamaa:** Appilikeeshiniiwwan React'f kutaalee `<ETNConnectButton />` kan 'plug-and-play' ta'an of keessatti hammata.
*   **Nageenyummaa Sadarkaa-Entarpiraayizii:** Deggarsa guutuu OIDC Authorization Code Flow kan PKCE fi Token Rotation ofumaan (automatic) qabu.
*   **Type-Safe:** Amanamummaa koodii fi bu'a-qabeessummaa misoomsitootaa mirkaneessuuf guutummaatti TypeScript dhaan barreeffame.
*   **Madda Banaa (Open Source):** Guutummaatti ifa kan ta'ee fi gumaacha hawaasaatiif GitHub irratti kan argamu.

### Akkamitti Jalqabamu

SDK'n kun hatattamaan karaa kuusaa npm uummataa (public npm registry) ni argama. Haa ta'u malee, layibiraaricha walitti-fiduu dura, misoomsitoonni hayyama API barbaachisaa ta'e (Client ID fi Client Secret) argachuuf appilikeeshinii isaanii galmeessisuu qabu.

1.  **Appii Keessan Galmeessisaa:** Hayyama misoomsitootaa keessan maddisiisuuf **[Waltajjii Akkaawuntii ETN](https://account.etnecosystem.org/)** daawwadhaa.
2.  **SDK Fe'adhaa:** Pirojektii keessan keessatti `npm install @etn-ecosystem/identity-sdk` ajaja jedhu kana fiigaa.
3.  **Ijaaraa:** Adeemsa seenuu (sign-in) keessan isa jalqabaa hojiirra oolchuuf sanada [GitHub](https://github.com/ETN-Ecosystem/ETN-Identity-SDK) irratti argamu ilaalaa.
