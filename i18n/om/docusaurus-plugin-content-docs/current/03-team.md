Sirni Ikkoo ETN garee hojii fi muuxannoo qabuun kan geggeeffamu yoo ta'u, kaayyoo isaa kan Afrikaa furmaataalee waloo hin taaneen humneessuu irratti xiyyeeffata. Gareen keenya teeknooloojii blokcheenii, AI, dizayinii, gabaa, fi hawaasa ijaaruu keessatti beekumsa walitti fida.

### Hogganummaa

*   **Jason Peters** – Hundeesaa Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** CEO ETHIO TECH Group LLC, Hundeesaa ETHIO TECH AI. Hogganaa fi hundeesaa mul'ata qabu, kallattii tooftaa fi guddina sirna ikkoo ETN geggeessu.

### Hundeesitoota Waliin fi Gorsitoota

*   **Nathnael Biruk** – Hundeesaa Waliin ETN
    *   **Duubee fi Gumaacha:** Hundeesaa CryptoTalk-ET. Mariiwwan kiriptoo fi hawaasa ijaaruu keessatti beekumsa isaa fayyadamuun guddina tooftaa ETN deeggara.
*   **Dawit Mengistu** – Gorsaa Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** Daayirektara Telegram Wallet Africa. Walsimsiisa waletii fi aplikeeshiniiwwan blokcheenii Afrikaa keessatti gorsa deeggarsa kenna.
*   **Nebiyu Sultan** – Gorsaa fi Dabalataa-Chain Dev Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** Endubis Wallet. Piroojektii ETN waletii bulchuu fi tooftaalee faayinaansii waloo hin taane (DeFi) keessatti beekumsa isaa fayyadamuun gorsa kenna.

### Garee Misoomaa

*   **Elias Taye** – Misoomaa Olaanaa Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** Misoomaa Bu'uuraa ETHIO TECH AI. Garee misoomaa geggeessa, AI walsimsiisuu fi furmaataalee blokcheenii irratti xiyyeeffata.
*   **Al Aymen** – Dizayinerii Weebii fi Misoomaa Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** Interface fayyadamaa dizayinii fi misooma, hojii salphaa fi muuxannoo fayyadamaa salphaa mirkaneessa.

### Giraafiksii fi Quunnamtii

*   **Biniam Samuel** – Dizayinerii Giraafiksii Olaanaa Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** Abbaa Qabeenyaa Waliin IDENTICO. Eenyummaa viizuwaliin ETN irratti hojjeta, qabeenya brandii irratti dizayiniiwwan hirmaannaa qaban uuma.
*   **Asnake Gizaw** – Ofiseera Quunnamtii Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** Miseensa Garee Gabaa Ethiopian Airlines. Quunnamtii fi hojiiwwan PR bulcha, walitti dhufeenya fi hirmaannaa ummataa jajjabeessa.

*   **Welid Sherif** – Qindeessaa Hawaasaa Sirna Ikkoo ETN
    *   **Duubee fi Gumaacha:** ET Netsa Apps hawaasa moodeereetara. Sirna ikkoo keessatti hirmaannaa fi fudhatama fayyadamaa jajjabeessuun sochiiwwan hawaasaa geggeessa.
