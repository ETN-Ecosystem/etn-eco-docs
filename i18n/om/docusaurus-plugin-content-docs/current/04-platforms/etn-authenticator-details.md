# ETN Authenticator

## Sirna Ikkoo Nageenya Isaa Cimsuu

ETN Authenticator sirna mirkaneeffannaa lamaa (2FA) kan of danda'e yoo ta'u, nageenya Sirna Ikkoo ETN guutuu cimsuuf qophaa'e dha. Sirni kunis "ETN Numbers NFTs" akka bu'uura mirkaneeffannaa isaa fayyadamuun adda ta'a. Dabalataanis, Wallet Connect fayyadamuun fayyadamtootaaf muuxannoo mirkaneeffannaa tokkoo kan sirna ikkoo guutuu keessatti argamu salphisa.

ETN Authenticator nageenyaa fi muuxannoo fayyadamaa lamaanuu keessatti fooyya'iinsa guddaa agarsiisa. Fayyadamni "ETN Numbers NFTs" tooftaa haaraa kan eenyummaa diigitaalii fi to'annoo argachuu agarsiisa, mirkaneeffannaa qabeenya diigitaalii addaa kan fayyadamaan qabu waliin walitti hidha. Kunis, tooftaalee 2FA kanneen durii kanneen yeroo baay'ee koodiiwwan aplikeeshinii irraa uumaman yookiin ergaawwan SMS kanneen rakkoolee adda addaaf saaxilamoo ta'an irra darba. 2FA gara NFTs on-chainitti hidhuun, mirkaneeffannaan hojii qabeenya diigitaalii addaa kan waletii kiriptoo nageenya qabuun mirkaneessuu ta'a, kunis bu'uuraalee Web3 kan bilisummaa fi to'annoo fayyadamaa waliin kan walsimu dha.

Walitti dhufeenyi Wallet Connect hojiiwwan Web3 kanneen yeroo ammaa jiran waliin kan walsimu yoo ta'u, kanneen waletiiwwan TON waliin hojjetanif tooftaa walitti dhufeenya beekamaa fi salphaa kenna. Kaayyoon "mirkaneeffannaa sirna ikkoo tokkoo" tooftaa salphaa, kan Galmee Tokkoo (SSO) fakkaatu agarsiisa. Sirni akkasii kunis, fayyadamtoonni madaalliiwwan ETN kanneen bal'achaa jiran, kanneen yeroo ammaa tajaajiloota adda addaa kudhan ol qaban, yeroo deeman rakkoo hir'isuuf murteessaa dha. Sochiin kunis, nageenya fooyyessuu qofa osoo hin taane, hojii fi gatii dhalootaa kanneen "ETN Numbers NFTs" jedhaman qabaachuu isaa dabala, kunis isaan qabeenya walitti qabaman yookiin meeshaalee bulchiinsaa irraa gara bu'uura eenyummaa diigitaalii fi haala nageenyaa fayyadamaa kan sirna ikkoo keessatti argamanitti jijjiiruu danda'a. Kunis, hawaasa fayyadamaa kan hirmaannaa qabu kan faayidaalee mul'atan kan hirmaannaa NFT sirna ikkoo ETN keessatti argaman beeku fi gatii kennu akka horatu gochuu danda'a, kunis fedhii NFTwwan addaa kanaaf uumuu danda'a.
