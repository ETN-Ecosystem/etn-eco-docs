ETN Bio madaallii tarreeffama biolinkii kan Linktree fakkaatu yoo ta'u, dabalataanis fayyadamtoonni waletii isaanii walsimsiisuu fi ETNn gorsa fudhachuu akka danda'an godha. Kunis fayyadamtootaaf profaayilii giddugaleessaa kan argamuu isaanii onlaayinii hunda agarsiisuuf kenna.
