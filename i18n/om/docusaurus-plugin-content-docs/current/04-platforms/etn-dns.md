ETN DNS sirna maqaa doomeenii waloo hin taane kan blokcheenii TON irratti ijaarame yoo ta'u, fayyadamtoonni maqaa doomeenii gadii addaa kan `.etnetsacoin.ton` jedhu jalatti akka NFTtti bitachuu fi uumuu dandeessisa, kunis abbummaa diigitaalii dhugaa kenna.
