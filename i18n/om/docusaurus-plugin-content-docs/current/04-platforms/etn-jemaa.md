# ETN Jem’aa

## Madaallii Miidiyaa Hawaasaa Web 3.0 Kan Ofii Keenyaa – Eegamaa Jira

ETN Jem'aa madaallii miidiyaa hawaasaa Web 3.0 kan sirna ikkoo kanaa ti. Kaayyoon isaa iddoo waloo hin taane kan walqunnamtii hawaasaa, qabiyyee walii qooduu, fi hirmaannaa kennuu dha, teeknooloojii blokcheenii fayyadamuun to'annoo fayyadamaa fi dhuunfaa fooyya'aa argachuuf.
