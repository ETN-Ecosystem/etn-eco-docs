# ETN PAY

## Karaa Kaffaltii $ETN Madaalliiwwan Hunda Irratti Fudhachuuf – Eegamaa Jira (Plugin WooCommerce ETN Learn Keessatti Walsimsiifamee fi Hojiiwwan Qorataa Jira)

ETN Pay furmaata walsimsiisa kaffaltii salphaa kan nama kamiyyuu kaffaltii ETN dabalachuu barbaaduuf. Kunis, API argachuu fi hojiiwwan faayinaansii kanneen aplikeeshiniiwwan galmaa'an tokko tokkoo fayyadamaaf hordofa, kanneen akka tajaajila invoysii fi mirkaneeffannaa on-chain dabalatee.
Oomishaalee gadii kanneen keessaa:
- Plugin WooCommerce kan dukaana onlaayinii
- **ETN Pay: Suq** – POS-ERP kan hojiiwwan baadiyyaa kanneen jalqaba irratti oflaayinii, maallaqa hin qabneef, kanis walitti qabama gibiraa ofumaan hojjetu.
