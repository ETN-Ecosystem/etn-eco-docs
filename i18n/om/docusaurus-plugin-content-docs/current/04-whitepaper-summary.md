**Waraqaan Adii Sirna Ikkoo ETN V1.2** mul'ata keenya, ergaa keenya, fi tooftaa bal'aa kan namootaa fi daldaltoota Afrikaa keessatti furmaataalee waloo hin taane kanneen blokcheenii TON irratti ijaaraman fayyadamuun humneessuuf qabnu ibsa.

### Mul'ata fi Ergaa

Sirni Ikkoo ETN jalqaba haaraa kan hojiiwwan dhugaa addunyaa irratti mul'atan kanneen diigitaalii fi aplikeeshiniiwwan waloo hin taaneef uumuuf qophaa'e dha. Mul'anni keenya inni olaanaan, sirna faayinaansii waloo hin taane fi hunda hammate kan bilisummaa faayinaansii fi dabalata diigitaalii miliyoonota Itoophiyaa fi Afrikaa bal'aa keessatti jajjabeessu hundeessuu dha. Kunis, tajaajiloota faayinaansii fi diigitaalii salphaatti argaman, iftoomina qaban, fi nageenya qaban kennuun hojii irra oola.

### Qabxiiwwan Bu'uuraa fi Furmaataalee

Onnee sirna ikkoo kanaa irratti **ET NETSA Coin ($ETN)** jira, kunis kiriptoo keenya kan tajaajiloota adda addaa humneessu dha. Sirni ikkoo kunis, sagantaalee bal'aa kanneen 17 hanga 27 tajaajiloota adda addaa qaban kennuuf qophaa'e dha, kanneen keessaa:
*   **ETN DNS:** Galmee maqaa doomeenii waloo hin taane.
*   **ETN Hosting:** Tajaajila weebii waloo hin taane.
*   **ETN Learn:** Barnoota blokcheenii onlaayinii.
*   **ETN Pay:** Hojii kaffaltii nageenya qabu.
*   **ETN Equb:** Sirna qusannoo Itoophiyaa kan durii, kan blokcheenii irratti fooyya'e.
*   **ETN Ads, ETN Sell, ETN Bio, ETN Council, ETN Home, ETN Join:** Fi kanneen biroo hedduu, kanneen hundi fedhii fi carraa adda addaa furu.

### Barbaachisummaa Blokcheenii TON Afrikaaf

Ijaarsi Sirna Ikkoo ETN blokcheenii The Open Network (TON) waliin walitti hidhata cimaa qaba. TON tooftaa qophaa'e kan filatame yoo ta'u, kunis faayidaalee adda addaa kanneen haala Afrikaaf baay'ee barbaachisoo ta'an qabaachuun isaa ti:
*   **Kaffaltii Transakshinii Gad-aanaa:** Transakshiniiwwan guyyaa guyyaa dinagdeedhaan hojii irra oolchu.
*   **Saffisa Transakshinii Ariifataa:** Muuxannoo fayyadamaa salphaa mirkaneessu.
*   **Waletiiwwan Fayyadamtootaaf Salphaa fi Walsimsiisa Telegram:** Fudhatama bal'aa Telegram Afrikaa keessatti argamu fayyadamuun fayyadamtoota salphaatti galchu.
*   **Bal'ina:** Sirni ikkoo kunis, fayyadamtoota guddachaa jiran salphaatti akka fudhatu mirkaneessu.

Walitti dhufeenyi kunis, fayyadamtoota naannoo kanneen blokcheenii barbaachisummaa ta'eef, sababa tajaajila faayinaansii idilee argachuun isaanitti ulfaatuuf, galchuuf murteessaa dha. Guddinni dinagdee diigitaalii Afrikaa fi jijjiirramni addunyaa gara Global Southitti argamu, iddoo tooftaa Sirna Ikkoo ETN caalaatti agarsiisa.

### Bulchiinsa fi Dandeettii Jiraachuu

Sirni Ikkoo ETN **sirna bulchiinsaa kan NFT qofa irratti hundaa'e** fayyadama, kunis humna murtoo kennuu kan abbummaa NFTwwan bulchiinsaa adda addaa (Soulbound Tokens, Council Minister NFTs, Investor NFTs) fi $ETN Treasury Bonds waliin kan walqabatu dha. Kunis, bulchiinsa hawaasaa kan geggeeffamu yoo ta'u, bulchiinsa misoomaa waliin walqabata.

Tooftaan dandeettii jiraachuu keenyaa, kaffaltii transakshinii yookiin tajaajila $ETN irratti, madaalliiwwan hunda irratti walitti qabuu irratti xiyyeeffata. Maallaqni walitti qabame kunis, waletiiwwan baay'ee-saayinii qaban keessatti nageenya qabuun kuufamee, waggaatti yeroo tokko tokkoo miseensota hawaasaaf raabsama. Tooftaan kunis, kan filatame yoo ta'u, kaayyoon isaa, tokeniin gatii isaa akka dabaluu fi fedhii dhalootaa akka horatu gochuun, gatii yeroo dheeraaf kanneen jalqaba irratti investii godhan fi hirmaatanif uumuu dha.

### Rakkoolee

Sirni Ikkoo ETN abdii guddaa qabaatus, rakkoolee kanneen akka guddina itti fufiinsaa fi tooftaa sirna ikkoo TON keessatti deemuuf maallaqa gahaa argachuu, kanneen piroojektoota tajaajila irratti xiyyeeffatan gidduu jiran, kanneen tooftaa hin taaneen guddachaa jiran, beekamtii akka argatan gochuu dabalata. Kutannoon keenya quunnamtii walirraa hin cinne fi gatii dhugaa agarsiisuu irratti xiyyeeffata.

Walumaagalatti, Sirni Ikkoo ETN sirna ikkoo Web3 kan sochii qabu fi dandeettii jiraachuu qabu kan furmaataalee waloo hin taane haaraa ta'an fi hubannoo fedhii naannoo qabu walitti fiduun, Afrikaa fuuldura waloo hin taane irratti argama.
