Tokeniin $ETN kiriptoo-saayinsii dhalootaa kan Sirna Ikkoo ETN humneessu yoo ta'u, hojiiwwan salphisuuf, hirmaannaa jajjabeessuuf, fi guddina furmaataalee waloo hin taane keenyaa geggeessuuf qophaa'e dha.

### Qabeenya Waliigalaa

Qabeenyi waliigalaa $ETN **5,000,000 ETN** irratti murtaa'eera.

*   **Qabeenya Socho'aa:** **624,808 ETN (12.50%)**
    *   Kunis, tokenoota yeroo ammaa jiran fi sirna ikkoo keessatti hojii irra jiran agarsiisa.

*   **Qabeenya Cufame/Qabame:** **4,375,192 $ETN (87.5%)**

### Raabsa fi Ramaddii Haaraa

Raabsi fi ramaddiin $ETN tooftaa qophaa'een kan fooyya'e yoo ta'u, kunis fedhiiwwan sirna ikkoo kanneen jijjiiramaa jiran fi tooftaa dandeettii jiraachuu isaa agarsiisa.



  *   **Fooyya'iinsa Tokoomiksii, Kan Kiriptoo-saayinsii Yeroo Hundaa Jijjiiramaa Jiru Irratti Hundaa'e. (Preesaleen hojiiwwan aplikeeshinii keessatti kan bakka buufame yoo ta'u, ETN AirDrop TWA Keessatti)**

    *   **Waletii AirDrop/Preesale:** **2,000,000 $ETN**
        *   Waletiin walitti makame kunis, tokenoota yeroo ammaa fi fuulduraaf raabsa hawaasaaf ramadaman kanneen akka airdropiiwwan tapha fakkaatan kanneen Telegram Mini Apps fayyadamuun raabsaman dabalata. Ramaddiin kunis, ramaddiiwwan airdropii fi preesale adda addaa kan durii bakka buusa.
    *   **Dev Fund:** **1,000,000 $ETN**
        *   Misooma itti fufiinsaa, qorannoo, fi sochiiwwan tooftaa Bu'uura ETN tiif qabameera.
      *   **LP Jalqabaa:** **2,000,000 $ETN**
        *  Warra jalqaba irratti amanan fi warra jalqaba irratti fudhatan investii isaanii akka baay'isan gochuuf gatii jalqabaa baay'ee gadi aanaa irraa jalqabe.

### Gatii Jalqabaa (Waraqaa Adii V1.1 irraa)

Gatiin jalqabaa $ETN kan murtaa'e, poolii liikiiditii keessatti walitti makamuu isaa ti.
*   **Poolii Liikiiditii:** 2,000,000 $ETN fi 200 TON walitti makame.
*   **Herrega:**
    *   Gatiin 1 $ETN = (200 TON) / (2,000,000 ETN) = 1/10,000 TON
*   Kunis, gatii jalqabaa tokenii $ETN tokkoo 1/10,000 TON tokenii irratti murteesse.

Ijaarsi tooftaa haaraa kunis, dandeettii jiraachuu yeroo dheeraa, hirmaannaa hawaasaa, fi ramaddii tooftaa qabeenyaa kan guddina fi hojii Sirna Ikkoo ETN jajjabeessu irratti xiyyeeffata.
