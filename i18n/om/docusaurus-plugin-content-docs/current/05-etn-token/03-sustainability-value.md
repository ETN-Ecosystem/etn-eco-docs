Sirni Ikkoo ETN tooftaa guddina cimaa fi dandeettii jiraachuu qabu irratti kan ijaarame yoo ta'u, kaayyoon isaa warra tokenii qaban fi hawaasaaf gatii yeroo dheeraa uumuu dha, tooftaa tokonomiksii tilmaamaa irraa fagaachuun.

### Gatii Tokenii Hir'isuu Diduu

Hawaasaa fi Mana Maree Bulchiinsa ETN waliin erga marii cimaa godhamee booda, **tooftaan gatii tokenii hir'isuu, kan waraqaa adii duraanii (fkn, V1.1) keessatti dhiyaate, didameera.** Tokenii gubuun mul'ata piroojektichaa kan yeroo dheeraa waliin kan hin walsimne ta'uu isaa murtaa'eera, keessattuu qabeenyi waliigalaa $ETN kan 5,000,000 ta'e duruu daangeffamee waan jiruuf. Mana Mareen gatii tokenii dabaluuf fi hirmaannaa jajjabeessuuf tooftaa tooftaa qophaa'e filate.

### Tooftaa Dandeettii Jiraachuu Ammaa

Tooftaan dandeettii jiraachuu keenyaa inni ammaa guddina dhalootaa, fedhii hojii irra ooluu irratti hundaa'e, fi raabsa gatii kallattii kanneen qabeenya qabanif irratti xiyyeeffata:

1.  **Kaffaltii Madaallii $ETNn Walitti Qabame:**
    *   Sirni Ikkoo ETN kaffaltii transakshinii yookiin kaffaltii tajaajilaa kan tajaajiloota isaa fi madaalliiwwan isaa hunda irratti hojii irra oolcha.
    *   Kaffaltiiwwan sochiiwwan kana irraa argaman hundi $ETN qofaadhaan walitti qabama.
    *   Tokenoonni $ETN walitti qabaman kunis, waletii baay'ee-saayinii kan madaallii tokko tokkoo keessatti nageenya qabuun kuufamu.

2.  **Raabsa Waggaa Kanneen Qabeenya Qabanif:**
    *   Dhuma waggaa faayinaansii tokkoo tokkoo irratti, maallaqni $ETN walitti qabame kan waletiiwwan baay'ee-saayinii kana irraa argame, kanneen qabeenya hawaasaa qabanif raabsama. Kunis, guddinni sirna ikkoo kanneen hirmaatan fi gumaachanif kallattiin akka fayyadu mirkaneessa.

3.  **Gatii Hojii fi Fedhii Irraa Uumuu:**
    *   Bakka hanqina namtolchee kan gubuun uumamuun, Sirni Ikkoo ETN gatii tokenii dabaluu irratti xiyyeeffata, kunis fedhii bal'aa fi fayyadama madaallii fayyadamuun. 
    *   Fayyadamtoonni $ETN walitti qabachuun tajaajiloota argachuuf, walitti dhufeenya waloo hin taane fayyadamuun, fedhii dhalootaa jajjabeessina fi qabeenya socho'aa dhalootaan hir'isna. Tooftaan kunis, rakkoolee yeroo gabaabaa kanneen tooftaa gubuu waliin walqabatan irraa fagaata fi fudhatama dhugaa jajjabeessa.

Tooftaan haaraa kunis, guddinni sirna ikkoo piroojektii fi hawaasa isaa lamaanuu akka fayyadu mirkaneessa, fedhii hunda kan milkaa'ina itti fufiinsaa fi gatii yeroo dheeraa kanneen jalqaba irratti investii godhan fi hirmaatanif uumuuf walitti fida.
