# Jechoota Bu'uuraa

Galmeen jechootaa kunis hiika jechoota bu'uuraa, madaalliiwwan, fi teeknooloojiiwwan kanneen Sirna Ikkoo ETN waliin walqabatan kenna.

*   **$ETN (ET Netsa Coin):** Kiriptoo-saayinsii hojii dhalootaa kan Sirna Ikkoo ETN, kan blokcheenii TON irratti ijaarame, kan hojiiwwan, kaffaltiiwwan, staking, bulchiinsa, fi badhaasaaf madaalliiwwan sirna ikkoo hunda irratti fayyadu.
*   **ETN Ads (ads.ETnetsaCoin.ton):** Portalii beeksisaa waloo hin taane kan Sirna Ikkoo ETN keessatti argamu kan daldaltootaa fi uumtoota oomishaalee fi tajaajiloota isaanii chaanaalota Telegram fi madaalliiwwan biroo kanneen $ETN fayyadamuun beeksifachuuf dandeessisu.
*   **ETN Auth:** Madaallii mirkaneeffannaa kan sirna ETN Numbers keessatti argamu, kan NFTwwan fayyadamuun eenyummaa mirkaneessuu fi to'annoo argachuu danda'u.
*   **ETN Authenticator:** Meeshaa yookiin aplikeeshinii kan hojii ETN Auth salphisuuf qophaa'e, kan NFTwwan mirkaneeffannaaf fayyadan yookiin akka tooftaa 2FAtti hojjetan bulchuu danda'u.
*   **ETN Bio (bio.ETnetsaCoin.ton):** Tajaajila biolinkii waloo hin taane kan blokcheenii TON irratti argamu, kan fayyadamtootaaf fuulawwan argamuu diigitaalii walitti qabaman uumuuf dandeessisu, kanneen hojiiwwan addaa $ETNn kaffalaman.
*   **ETN Council (dao.ETnetsaCoin.ton):** Madaallii bulchiinsaa kan sirna ikkoo ETN, kan murtoo kennuu kan hawaasaan geggeeffamu kan $ETN Treasury Bonds fi NFTwwan Bulchiinsaa addaa fayyadamuun dandeessisu.
*   **ETN DNS (ETN Domain Name System):** Sirna Maqaa Doomeenii kan blokcheenii irratti hundaa'e kan fayyadamtootaaf maqaa doomeenii gadii kan .etnetsacoin.ton jedhu jalatti akka NFTtti bitachuu fi uumuu dandeessisu, kan hojiiwwan $ETN fayyadamuun.
*   **ETN Equb:** Fooyya'iinsa blokcheenii kan sirna faayinaansii Itoophiyaa kan durii "Equb" (waldaan qusannoo fi liqii naanna'aa), kan waletiiwwan TON Multisig fi $ETN gumaachaaf fi kaffaltiiwwanif fayyadamu.
*   **ETN Gebeya (duraan ETN Sell - sell.ETnetsaCoin.ton):** Madaallii e-commerce kan meeshaalee virtualii fi qaamaa kanneen $ETN tokenoota fayyadamuun bitachuu fi gurguruu deeggara.
*   **ETN Gulit:** Gabaa NFT kan sirna ETN Numbers keessatti argamu, kan NFTwwan ETN Numbers fi NFTwwan biroo kanneen TON irratti hundaa'an daldaluuf kan yaade, kan hojiiwwan $ETN fayyadamuun.
*   **ETN Home (home.ETnetsaCoin.ton):** Weebsaayitii ofiisaa fi giddugala sirna ikkoo ETN, kan madaalliiwwan hundaaf, oduuwwan, fi qabeenyaaf argachuu dandeessisu.
*   **ETN Hosting:** Tajaajila weebii waloo hin taane kan blokcheenii TON irratti TON Storage fayyadamuun kennamu, kan kaffaltiiwwan $ETNn godhaman.
*   **ETN Join (join.ETnetsaCoin.ton):** Madaallii aplikeeshinii kan piroojektoota gurguddoo kanneen sirna ikkoo ETN waliin walsimsiisuu yookiin makamuu barbaadanif, kan TON Connect fayyadamuun warra walitti dhufeenya qaban galchuuf.
*   **ETN Learn (learn.ETnetsaCoin.ton):** Madaallii barnootaa onlaayinii kan Web3 waliigalaa, blokcheenii, fi barnoota kiriptoo-saayinsii irratti xiyyeeffatu, kan kaayyoo galmee addaa qabu. Barsiisonni $ETNn kaffalamu, fi fayyadamtoonni barnoota addaa $ETNn kaffaluu danda'u.
*   **ETN Numbers:** Walitti qabama NFT kan blokcheenii TON irratti argamu, kan madaalliiwwan gadii ETN Gulit, ETN Auth, fi ETN Authenticator waliin walitti qabama uumu.
*   **ETN Pay:** Furmaata hojii kaffaltii nageenya qabu kan sirna ikkoo ETN keessatti $ETN fayyadamuun. Plugin WooCommerce kan daldaltoota $ETN fudhachuu dandeessisu dabalata.
*   **Afaan Oromoo:** Afaan Kuushitii kan Itoophiyaa keessatti dubbatamu, kan jechi "Netsa" (bilisa, birmadummaa, walabummaa jechuun hiikamu) irraa dhufe.
*   **DeFi (Faayinaansii Waloo Hin Taane):** Sirna faayinaansii kan teeknooloojii blokcheenii irratti ijaarame kan giddugaleessa tokko malee hojjetu, kan hojiiwwan namaa fi namaa gidduu fi tajaajila faayinaansii dandeessisu.
*   **GekoTerminal:** Madaallii hordoffii fi odeeffannoo kiriptoo-saayinsii kan $ETN irratti tarreeffame.
*   **Warra Eegduu Itoophiyaa:** Walitti qabama NFT kanneen super hero Itoophiyaa 20, kanneen tokko tokkoon madaallii sirna ikkoo ETN eegan fi warra qabanif qooda bu'aa madaallii sanaa kennan.
*   **Jetton:** Sadarkaa tokenoota fungible kan blokcheenii TON irratti argaman, kan $ETN waliin walsimu.
*   **NFT (Non-Fungible Token):** Qabeenya diigitaalii addaa kan abbummaa meeshaa addaa, kutaa qabiyyee, yookiin mirga blokcheenii irratti agarsiisu. ETN DNS, Warra Eegduu Itoophiyaa, fi ETN Numbers keessatti fayyadama.
*   **Netsa (ነፃ):** Jecha Amaariffaa kan "bilisa," "birmadummaa," yookiin "walabummaa" jechuun hiikamu, kan falaasama bu'uuraa Sirna Ikkoo ETN agarsiisu.
*   **ROSCA (Rotating Savings and Credit Association):** Sirna faayinaansii idilee kan durii kan gareen tokko yeroo hundaa maallaqa walitti qabu, kanneen miseensotaaf tokko tokkoon kennamu. ETN Equb fooyya'iinsa blokcheenii kan kanaa ti.
*   **SBT (Soulbound Token):** Akaakuu NFT kan hin jijjiiramne kan waletii addaa waliin walqabatu, yeroo baay'ee eenyummaa, milkaa'ina, yookiin kaayyoo bulchiinsaa agarsiisuuf fayyadama. ETN Council fi ETN Learn keessatti fayyadama.
*   **Tact:** Afaan sagantaa kan smart contractiiwwan blokcheenii TON irratti barreessuuf fayyadu, kan walitti dhufeenya tokenii $ETN waliin walqabatee ibsame.
*   **Telegram:** Madaallii ergaa beekamaa kan fayyadamtoota baay'ee Afrikaa keessatti qabu. Waletiin TON ofiisaa Telegram keessatti walsimsiifameera, kunis fayyadamtoota gara tajaajiloota TON irratti hundaa'anitti galchuuf salphisa.
*   **TON Blockchain (The Open Network):** Blokcheenii Layer-1 kan saffisaa, nageenya qabu, fi bal'aa kan bu'uuraalee sirna ikkoo ETN guutuu ta'ee tajaajilu.
*   **TON Connect SDK:** Meeshaa Misoomaa Sooftiweerii kan aplikeeshiniiwwan waloo hin taane (dApps) fi weebsaayitiiwwan waletiiwwan TON waliin nageenya qabuun walitti hidhachuu dandeessisu, kan mirkaneeffannaa fi saayinii transakshinii. Madaalliiwwan ETN adda addaa fayyadamu.
*   **TON DNS:** Sirna maqaa doomeenii waloo hin taane kan blokcheenii TON keessatti walsimsiifame, kan ETN DNS irratti ijaarame.
*   **TON Multisig Wallet:** Akaakuu waletii kiriptoo-saayinsii kan blokcheenii TON irratti argamu kan saayinii baay'ee (hayyama) kanneen abbootii qabeenyaa murtaa'an irraa barbaadu, kan hojiiwwan mirkaneessuuf. ETN Equb fi bulchiinsa kaffaltii madaallii tiif fayyadama.
*   **TON Storage:** Sirna kuusaa faayilii waloo hin taane kan blokcheenii TON irratti ijaarame, kan ETN Hosting fayyadamu.
*   **TonAPI:** Walitti qabama APIwwan HTTP sadarkaa olaanaa kan odeeffannoo indeksii kan blokcheenii TON irraa kennan, kan ETN Pay kaffaltiiwwan fi mirkaneeffannaa transakshinii tiif fayyadu.
*   **Web3:** Yaada dhaloota interneetii itti aanuu, kan waloo hin taane, teeknooloojiiwwan blokcheenii, dinagdee tokenii irratti hundaa'e, fi to'annoo fayyadamaa guddaa kan odeeffannoo fi eenyummaa irratti qabu. Sirni Ikkoo ETN sochii Web3 ti.
*   **Ahadu Badge:** Soulbound Token (SBT) yaadannoo kan blokcheenii TON irratti uumame, kan eebbifamtoota jalqabaa ("Ahadu Batch") ETN Learnif kenname. Barnoota ETN Learn hundaaf yeroo jireenyaa guutuu argachuu dandeessisa fi ragaa xumura sagantaa ta'ee tajaajila.
*   **$ETN Treasury Bonds:** Qabeenya bulchiinsaa kan Mana Maree ETN keessatti argamu, kan qabeenya waliigalaa 200 qabu. Tokkoon tokkoon bondii sagalee 1 kenna. Tooftaa bulchiinsaa kan NFT qofa irratti hundaa'e keessatti, kunneen akka qabeenya tokenii mirkanaa'aa (akka NFTwwan) kan sagalee kennuuf fayyadanitti hubatamu.
*   **ETN Authenticator:** Sirna mirkaneeffannaa lamaa (2FA) kan of danda'e kan Sirna Ikkoo ETN, kan ETN Numbers NFTs fi Wallet Connect fayyadamuun mirkaneeffannaa tokkoo fi nageenya qabu kan sirna ikkoo guutuu keessatti argamuuf.
*   **ETN Numbers NFTs:** Non-Fungible Tokens addaa kan ETN Authenticator tooftaa nageenyaa fi mirkaneeffannaa isaa keessatti fayyadu.
*   **Mesewer Books:** Madaallii haaraa kan Sirna Ikkoo ETN keessatti beeksifame. Odeeffannoon addaa waa'ee hojii isaa beeksisa dabalataa eegaa jira.
*   **Bulchiinsa NFT Irratti Hundaa'e:** Tooftaa murtoo kennuu kan Sirna Ikkoo ETN kan mirga sagalee kennuu kan abbummaa Non-Fungible Tokens addaa (SBTs, Council Minister NFTs, Investor NFTs dabalatee) fi $ETN Treasury Bonds (kanneen akka qabeenya tokenii mirkanaa'aa kan sagalee kennuuf fayyadan) waliin walqabatu. Kunis, bulchiinsa kan tokenoota $ETN waliigalaa, fungible qabachuu irratti hundaa'e irraa adda.
