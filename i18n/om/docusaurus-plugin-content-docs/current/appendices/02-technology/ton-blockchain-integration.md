# Walsimsiisa Blokcheenii TON

Sirni Ikkoo ETN Blokcheenii The Open Network (TON) irratti kan ijaarame yoo ta'u, hojiiwwan fi dandeettiiwwan isaa kanneen fooyya'an fayyadama.

## Ilaalcha Waliigalaa

### Blokcheenii TON
- Hojii olaanaa
- Ijaarsa bal'aa
- Baasii transakshinii gadi aanaa
- Hojiiwwan fooyya'an

### Faayidaalee Walsimsiisaa
- Transakshiniiwwan saffisaa
- Hojii irra oolmaa baasii
- Nageenya
- Bal'ina

## Walsimsiisa Teeknikaa

### Ijaarsa
- Walitti dhufeenya blokcheenii TON
- Hojii irra oolmaa smart contractii
- Walitti makamuu odeeffannoo
- Bulchiinsa haala

### Qabxiiwwan
- Walsimsiisa node
- Walitti dhufeenya API
- Walqunnamtii contractii
- Bulchiinsa yeroo

## Hojiilee

### Hojii Transakshinii
- Hojii saffisaa
- Kaffaltii gadi aanaa
- Dandeettii guddaa
- Mirkaneeffannaa amanamaa

### Smart Contractiiwwan
- Contractiiwwan TON waliin walsiman
- Hojii ofumaan hojjetu
- Hojii irra oolmaa nageenya qabu
- Odeeffannoo yeroo hundaa

## Misooma

### Meeshaalee
- Meeshaa misoomaa TON
- Tooftaalee qorannoo
- Meeshaalee hojii irra oolchuuf
- Sirna hordoffii

### Hojiilee Gaarii
- Sadarkaa koodii
- Tarkaanfiiwwan nageenyaa
- Hojiilee qorannoo
- Galmee

## Nageenya

### Tarkaanfiiwwan
- Nageenya networkii
- Odiitii contractii
- To'annoo argachuu
- Eegumsa odeeffannoo

### Hordoffii
- Haala networkii
- Hordoffii transakshinii
- Beeksisa nageenyaa
- Safartuu hojii

## Muuxannoo Fayyadamaa

### Interface
- Dizayinii fayyadamaa salphaa
- Navigeeshinii salphaa
- Transakshiniiwwan saffisaa
- Yaada ifa ta'e

### Walsimsiisa
- Walitti dhufeenya waletii
- Saayinii transakshinii
- Odeeffannoo haala
- Sirreeffama dogoggoraa

## Misooma Fuulduraa

### Hojiilee Karoorfaman
- Walsimsiisa fooyya'aa
- Dandeettiiwwan haaraa
- Fooyya'iinsa hojii
- Muuxannoo fayyadamaa

### Karaa Misoomaa
- Fooyya'iinsa teeknikaa
- Dabalata hojii
- Fooyya'iinsa nageenyaa
- Bal'ina madaallii

## Deeggarsa

### Galmee
- Qajeelcha walsimsiisaa
- Qajeelcha API
- Hojiilee gaarii
- Rakkoo furuu

### Qabeenya
- Meeshaalee misoomaa
- Naannoo qorannoo
- Deeggarsa hawaasaa
- Gargaarsa teeknikaa
