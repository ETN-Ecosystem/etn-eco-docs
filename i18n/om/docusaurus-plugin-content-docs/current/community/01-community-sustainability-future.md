# Hawaasa, Dandeettii Jiraachuu, fi Ilaalcha Fuulduraa

Milkaa'inni yeroo dheeraa Sirna Ikkoo ETN dandeettii isaa kan hawaasa sochii qabu horachuu, tooftaa hojii dandeettii jiraachuu qabu eeguu, fi rakkoolee Web3 kanneen jijjiiramaa jiran keessa darbuu irratti hundaa'a.

## Hirmaannaa Hawaasaa fi Sochiilee Guddinaa

Sirni Ikkoo ETN hawaasa isaa bu'uura falaasamaa fi tooftaa hojii isaa irratti kaa'a. Waraqaan Adii $ETN jalqaba irratti baay'ina hawaasaa miseensota 24,000 ol ta'an kan ibse yoo ta'u, kan Amaariffaa immoo 26,000 ol ta'uu isaa ibsuun, hordoffii jalqabaa guddaa agarsiisa. Sochiilee adda addaa kanneen hirmaannaa fi guddina sochii qabu jajjabeessuuf qophaa'an:

*   **Hojii Barnootaa:** "Ultimate Netsa Bootcamp," sochii guddaa kan ETN Learn, barattoota kutaa 12ffaa Itoophiyaa 7,000 ol ta'an milkaa'inaan hirmaachisuun Web3, blokcheenii, fi TON isaan barsiiseera. Kunis, barnoota bu'uuraa irratti xiyyeeffannoo guddaa qabaachuu agarsiisa.
*   **Qabiyyee fi Deeggarsa Walqunnamtii:** Tamsaasa kallattii Telegram irratti yeroo hundaa kan qorannoo gabaa, mallattoolee bittaa/gurgurtaa yeroo dhugaa, fi madaallii quunnamtii ifa ta'e kennuuf karoorfameera, kunis hirmaannaa sochii qabu jajjabeessa.
*   **Sagantaalee Jajjabeessaa:** Airdropiiwwan $ETN, carraawwan staking kanneen tooftaalee badhaasaa qaban, fi sochiiwwan hawaasaa adda addaa kanneen hirmaannaa fi gumaacha jajjabeessuuf karoorfaman. Tapha airdropii Telegram kan Itoophiyaa irratti hundaa'e, kan Ebla 2025tti jalqabame, fayyadamtootaaf ET Nesta coins ($ETN) tapha walqunnamtii qabuun akka argatan dandeessiseera, kunis elementoota aadaa fi badhaasa kiriptoo walitti makuun.
*   **Misooma Hawaasaan Geggeeffamu:** Madaallii ETN Ads kan hawaasa keessaa miseensa tokkoon kan injifatame yoo ta'u, kunis karaa gumaacha hawaasaa kan guddina oomishaa kallattiin dhiibbaa gochuu danda'u agarsiisa.
*   **Mallattoolee Fakkeenya:** Kennaa waliigalaa tokenoota $ETN kan hawaasaa kan Pavel Durovif kenname, tooftaa hirmaannaa fi fedhii addaa agarsiisa.
*   **Hojii Waliin Ifa:** Piroojektii kunis GitHub irratti argamuu isaa itti fufa, kunis gumaacha hawaasaa kanneen akka forks, pull requests, fi gabaasa rakkoo kanneen kutaa open-source isaa tiif jajjabeessa.

Tooftaan hawaasaa kan baay'ee walxaxaa ta'e kunis barnoota, jajjabeessaa faayinaansii, sochiiwwan walqunnamtii, fi carraawwan uumamaa waliin walitti fida. Sochiilee barnootaa kanneen akka Bootcamp fi madaallii ETN Learn bal'aa kanneen beekumsa bu'uuraa fi dandeettii hawaasa keessatti ijaaruuf kaayyoo qaban. Jajjabeessaa faayinaansii, kanneen akka airdropiiwwan, badhaasa staking, tooftaa qooda bu'aa kan "Warra Eegduu Itoophiyaa" NFTs irraa argamu, fi raabsa waggaa kan kaffaltii madaallii kanneen qabeenya qabanif, fedhii faayinaansii hawaasaa fi milkaa'ina sirna ikkoo waliin walsimsiisuuf tajaajilu. Hirmaannaan walqunnamtii sochiiwwan kanneen akka tapha Telegram fi sagantaalee qorannoo gabaa kallattii kanneen karoorfaman fayyadamuun jajjabeefama. Dabalataanis, hawaasa kallattiin hojii misoomaa keessatti hirmaachisuun, akka ETN Ads hackathon irratti mul'ate, fi bulchiinsa keessatti, Mana Maree ETN fayyadamuun, miira abbummaa cimaa fi kaayyoo waloo jajjabeessa. Tooftaan bal'aa kunis hawaasa kan hirmaannaa fi kutannoo qabu kan akka fayyadamtoota hin sochooneetti qofa osoo hin taane, akka gumaachitoota sochii qaban, deeggartoota, fi kanneen qabeenya qaban kan imala sirna ikkoo keessatti argamanitti hojjetu horachuuf qophaa'e dha.

## Tooftaa Dandeettii Jiraachuu Sirna Ikkoo ETN

Sirni Ikkoo ETN tooftaa dandeettii jiraachuu addaa kan gatii yeroo dheeraa uumuu fi fayyadamtoota hawaasaa isaa kanneen qabeenya qabanif kallattiin fayyaduu irratti xiyyeeffatu fudhateera, kan tooftaalee tokenii gubuu kanneen baay'ee beekamoo fi yeroo tokko tokko kanneen ceepha'aman irraa fagaachuun.

*   **Tooftaa Tokenii Gubuu Diduu:** Erga hawaasaa fi Mana Maree Bulchiinsa ETN waliin marii cimaa godhamee booda, **tooftaan gatii tokenii hir'isuu, kan waraqaa adii duraanii ($ETN Whitepaper V1.1) keessatti dhiyaate, didameera.** Sababni isaas, qabeenyi waliigalaa $ETN duruu daangeffamee waan jiruuf, tokenii gubuun gatii hin qabne yookiin kaayyoon isaa inni guddaan gabaa yeroo gabaabaa jajjabeessuu qofa akka ta'eetti ilaalamuu danda'a, osoo gatii bu'uuraa hin agarsiisin. 
*   **Galii Kaffaltii Irratti Hundaa'e:** Bu'uura tooftaa dandeettii jiraachuu kanaa, kaffaltii transakshinii yookiin kaffaltii tajaajilaa kan tajaajiloota fi madaalliiwwan isaa adda addaa hunda irratti hojii irra oolchuu irratti argama. Kunis, fayyadamtoonni ETN Ads, ETN Gebeya, ETN Hosting, ETN Pay, fi kanneen biroo waliin yeroo walqunnaman, kaffaltii xiqqaa, kan $ETNn kaffalamu, walitti qabama.
*   **Walitti Qabama Maallaqa Waloo Hin Taane:** Tokenoonni $ETN kanneen kaffaltiiwwan kana irraa walitti qabaman, waletiiwwan baay'ee-saayinii kan madaallii tokko tokkoo keessatti nageenya qabuun kuufamu. Kunis, iftoominaa fi nageenya galii walitti qabamee mirkaneessa.
*   **Raabsa Waggaa Kanneen Qabeenya Qabanif:** Dhuma waggaa faayinaansii tokkoo tokkoo irratti, maallaqni $ETN waletiiwwan madaallii tokko tokkoo kana irraa walitti qabame, kanneen qabeenya hawaasaa qabanif raabsama. Kunis, tooftaa qooda bu'aa kallattii uuma.
*   **Xiyyeeffannoo Fedhii Hojii Irratti Hundaa'e:** Bakka hanqina namtolchee kan gubuun uumamuun, sirni ikkoo kunis gatii tokenii dabaluu irratti xiyyeeffata, kunis fedhii bal'aa fi fayyadama madaallii fayyadamuun. Fayyadamtoota $ETN walitti qabachuun tajaajiloota argachuuf, walitti dhufeenya waloo hin taane fayyadamuun, fedhii dhalootaa jajjabeessina fi qabeenya socho'aa dhalootaan hir'isna. Tooftaan kunis, rakkoolee yeroo gabaabaa kanneen tooftaa gubuu waliin walqabatan irraa fagaata fi fudhatama dhugaa jajjabeessa.

Tooftaan haaraa kunis, guddinni sirna ikkoo piroojektii fi hawaasa isaa lamaanuu akka fayyadu mirkaneessa, fedhii hunda kan milkaa'ina itti fufiinsaa fi gatii yeroo dheeraa kanneen jalqaba irratti investii godhan fi hirmaatanif uumuuf walitti fida.

## Rakkoolee Adda Baafaman fi Tooftaalee Furmaataa

Sirni Ikkoo ETN, kan fedhii guddaa fi haaraa ta'e, rakkoolee adda addaa kanneen guddina isaa fi milkaa'ina yeroo dheeraa isaa irratti dhiibbaa gochuu danda'an beeka. Rakkooleen kunis kan keessoo, kan qabeenya argachuu waliin walqabatan, fi kan alaa, kanneen tooftaa gabaa kiriptoo-saayinsii bal'aa waliin walqabatan.

*   **Maallaqa Gahaa Argachuu:** Guddina itti fufiinsaa, qulqullina, fi bal'ina sirna ikkoo kan baay'ee walxaxaa ta'e akka ETN, qabeenya faayinaansii guddaa fi itti fufiinsaa qabu barbaada. Kunis, akka rakkootti ifatti adda baafameera.

    *   **Tooftaa Furmaataa:** Piroojektii kunis maallaqa argachuuf karaa adda addaa fayyadamuu barbaada, kanneen akka deeggarsa hawaasaa cimaa jajjabeessuu (kan gumaacha tokenii yookiin gumaacha kallattii ta'uu danda'u) fi walitti dhufeenya tooftaa kanneen investii yookiin qabeenya waloo fidan hundeessuu. Gurgurtaan NFTwwan Bulchiinsaa (Council Minister fi Investor NFTs kan TON coinif) tooftaa maallaqa sirna ikkoo kanaaf argachuuf ta'a.

*   **Furmaataalee Yeroo Dheeraa Sirna Ikkoo TON Keessatti Dursa Kennuu:** Blokcheenii TON, akka kanneen biroo hedduu, guddina sochiiwwan tilmaamaa, kanneen akka tokenoota meme baay'isuu, muuxateera. Kunis yeroo tokko tokko piroojektoota, akka ETN, kanneen hojii guddaa, yeroo dheeraa fi furmaataalee dhugaa addunyaa irratti xiyyeeffatan, dhiibbaa gochuu danda'a. Haala akkasii keessatti mul'achuu fi qabeenya harkisuun rakkoo guddaa dha.

    *   **Tooftaa Furmaataa:** Sirni Ikkoo ETN piroojektoota hojii irratti hundaa'an barbaachisummaa isaanii beeksisuuf karoorfata. Kunis, gatii fi dhiibbaa mul'atu kan madaalliiwwan isaa yeroo hundaa agarsiisuu fi hawaasa TON bal'aa waliin walqunnamuu, kanneen tooftaalee maallaqa isaa dabalatee, kanneen tooftaa hin taaneen guddachaa jiran irraa gara furmaataalee guddina itti fufiinsaa fi rakkoo dhugaa addunyaa furuu danda'anitti xiyyeeffannoo fi deeggarsa guddaa akka argatan jajjabeessuuf.

*   **Rakkoolee Argachuu Gabaa fi Hanqina Bu'uuraalee Misoomaa Furuu (Rakkoo Hin Mul'anne):** ETN Pitch Deck furmaataalee isaa haala rakkoolee kanneen akka Itoophiyaa keessatti mul'atan, kanneen akka hanqina karaa kaffaltii onlaayinii, daldala e-commerce addunyaa irratti argachuun ulfaachuu, fedhii bulchiinsa waloo hin taane fi barnootaa, fi carraawwan galii gahaa hin taane kanneen qabiyyee uumanif, keessatti kaa'a.

    *   **Tooftaa Furmaataa (Hin Mul'anne):** Sagantaan guutuu kan madaalliiwwan Sirna Ikkoo ETN (ETN Pay, ETN Gebeya, ETN Council, ETN Learn, ETN Ads, fi kanneen biroo) furmaata guutuu kan rakkoolee adda baafaman kanaaf qophaa'e dha. Madaalliiwwan tokko tokkoon kutaa addaa kan hanqina bu'uuraalee misoomaa fi argachuu kanaa furuuf xiyyeeffata.

Rakkoon alaa inni guddaan kan Sirna Ikkoo ETN, haala kiriptoo-saayinsii bal'aa fi sochii isaa kan hin murtoofne fi fedhii tilmaamaa isaa keessa deemuudha. Piroojektii kunis rakkoo kanneen sochiiwwan hojii irratti xiyyeeffatan, kanneen akka ETN, kanneen piroojektoota tilmaamaa guddaa qaban, kanneen akka tokenoota meme, kanneen marii fi sochii maallaqaa sirna ikkoo keessatti to'achuu danda'an, waliin dorgomuu qaban, beeka. Tooftaan furmaataa piroojektichaa kan ibsame—hojii isaa beeksisuuf, gatii mul'atu agarsiisuuf, fi hawaasa TON bal'aa waliin walqunnamuun xiyyeeffannoo gara furmaataalee yeroo dheeraa, dhiibbaa qabanitti jijjiiruuf—hubannoo dhugaa gabaa kanaa agarsiisa. Kunis, qulqullinni teeknikaa fi kenniinsi hojii dhugaa addunyaa irratti mul'atu, barbaachisaa ta'us, yeroo hundaa dhiibbaa fi deeggarsa barbaachisaa argachuuf gahaa akka hin taane, gabaa kan yeroo baay'ee hype fi tooftaalee yeroo gabaabaan geggeeffamu keessatti, agarsiisa. Kanaafuu, milkaa'inni dhumaa Sirna Ikkoo ETN dandeettii isaa kan tooftaa misoomaa isaa kan fedhii guddaa qabu hojii irra oolchuu irratti qofa osoo hin taane, dandeettii isaa kan mul'ata yeroo dheeraa isaa, dhiibbaa hawaas-dinagdee isaa, fi gatii dandeettii jiraachuu isaa kan hawaasa TON bal'aaf, akkasumas warra investii godhan, warra walitti dhufeenya qaban, fi fayyadamtoota addunyaa irratti argamanif ibsuu irratti hundaa'a.
