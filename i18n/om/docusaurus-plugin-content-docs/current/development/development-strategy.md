# Misooma fi Tooftaa Sirna Ikkoo

Misooma Sirna Ikkoo ETN tooftaa dursa kennuu ifa ta'e fi karaa misoomaa waggaa baay'ee kan bal'aa ta'een kan geggeeffamu yoo ta'u, kunis tooftaa hojii kan tajaajiloota isaa bal'aa ijaaruuf qophaa'e agarsiisa.

## Tooftaa Dursa Kennuu Madaallii

Sirni Ikkoo ETN tooftaa tooftaa kan madaalliiwwan isaa adda addaa hojii irra oolchuuf kan fudhatu yoo ta'u, kunis dandeettii isaanii kan fudhatama fayyadamaa saffisaa fi hirmaannaa itti fufiinsaa qabuuf qaban irratti hundaa'uun dursa kenna. Haalli addaa kunis galmee adda addaa keessatti jijjiirama xiqqaa agarsiisuu danda'a, garuu tartiibni waliigalaa ni mul'ata. Yeroo baay'ee, madaalliiwwan hojii battalaa guddaa fi fedhii gabaa qaban dura dursa kennamu. Tartiibni kunis waliigalaan akka armaan gadiitti mul'ata:

1. ETN Ads (ads.ETnetsaCoin.ton)
2. ETN Gebeya (duraan ETN Sell - sell.ETnetsaCoin.ton)
3. ETN Learn (learn.ETnetsaCoin.ton)
4. ETN Bio (bio.ETnetsaCoin.ton)
5. ETN Council (dao.ETnetsaCoin.ton)
6. ETN Home (home.ETnetsaCoin.ton)
7. ETN Join (join.ETnetsaCoin.ton)

Madaalliiwwan kanneen akka ETN Equb, ETN Hosting, ETN DNS, fi ETN Pay dabalataanis akka dursa guddaatti adda baafamaniiru, garuu sadarkaan isaanii addaa kan tarree guddaa keessatti argamu yeroo hundaa ifatti hin ibsamne. Karaa misoomaa kan galmee keessatti kennaman tartiiba misoomaa fi jalqabaa kan madaalliiwwan weebii bu'uuraa ifa ta'e kenna.

Tooftaan dursa kennuu kunis kaayyoo kan jalqaba irratti hojii bu'uuraa ijaaruu fi dhiibbaa networkii sirna ikkoo keessatti horachuu agarsiisa. Madaalliiwwan kanneen duraan baasuuf karoorfaman, kanneen akka ETN Ads, ETN Gebeya (Sell), fi ETN Learn, kanneen fayyadamtoota saffisaan harkisuu, baay'ina hojii $ETN uumuu, fi gatii bu'uuraa sirna ikkoo kanaa agarsiisuu danda'an dha. ETN Ads fi ETN Gebeya kallattiin fedhiiwwan daldalaa mul'atan kanneen beeksisaa fi e-commerce keessatti argaman furu, kunis hojiiwwan dhugaa kan tokenii $ETN tiif kennu. ETN Learn beekumsa qabu kan fayyadamtootaa ijaaruu keessatti ga'ee murteessaa qaba, namoota akkamitti kutaa sirna ikkoo kanaa adda addaa hojii irra oolchuu fi fayyadamuu akka danda'an barsiisa. Madaalliiwwan jalqabaa kunis gatii battalaa uumuuf kan qophaa'an yoo ta'u, fayyadamtoota dhalootaan harkisuuf carraan isaanii guddaa dha. Gama biraan, madaalliiwwan kanneen akka ETN Council (bulchiinsaaf) yookiin ETN Join (walsimsiisa piroojektoota alaa tiif) fayyaa yeroo dheeraa, waloo hin taane, fi bal'ina sirna ikkoo kanaaf murteessoo dha. Haa ta'u malee, hojii irra oolmaa fi fudhatamni isaanii milkaa'aa ta'uun yeroo baay'ee jiraachuu hawaasa hirmaannaa qabu fi tajaajiloota hojii irra oolan kanneen hundaa'an irratti hundaa'a. Kanaafuu, dursa kennuun kunis loojikaalii fakkaata: tooftaan isaa, jalqaba irratti tajaajiloota bu'uuraa fi baay'ina fayyadamtootaa murteessaa ta'e hundeessuu, sana booda tooftaalee bulchiinsaa kanneen fooyya'an fi tooftaalee hojii waliin alaa fi bal'inaaf qophaa'an sadarkaan ijaaruu dha.

## Karaa Misoomaa Bal'aa (2024-2027)

Sirni Ikkoo ETN karaa misoomaa waggaa sadii kan bal'aa ta'e kan bara 2024 hanga 2027tti bal'atu qaba. Karaan misoomaa kunis, karoora bal'aa, ji'a-ji'aan kan misooma madaallii, sadarkaa qorannoo beta, jalqaba ummataa, fi fooyya'iinsa hojii fi walsimsiisa itti aanu kenna. Karoorri bal'aa kunis, jijjiirama guddaa kan karaa misoomaa sadarkaa sadii (Jalqaba, Misooma, Guddina) kan waraqaa adii $ETN jalqabaa keessatti ibsame irraa kan agarsiisu yoo ta'u, kunis guddina guddaa kan karoora piroojektii fi tooftaa ibsa.

### Waggaa 1 (2024-2025):

Waggaa jalqabaa keessatti xiyyeeffannoon inni guddaan, madaalliiwwan dhiibbaa guddaa qaban kanneen jalqabaa jalqabuu irratti. Kunis, misooma, qorannoo beta, fi jalqaba ummataa kan ETN Ads fi ETN Gebeya (Sell) dabalata. Yeroo tokkootti, hojiin misoomaa jalqabaa kan ETN Learnis jalqabuuf karoorfameera.

### Waggaa 2 (2025-2026):

Waggaa lammaffaan hojiiwwan fi dandeettiiwwan madaalliiwwan duruu jalqabaman (ETN Ads fi ETN Gebeya) bal'isuuf kan qophaa'e dha. Waggaa kana keessatti jalqabni ummataa guutuu kan ETN Learnis ni mul'ata. Hojiin misoomaa kan ETN Bio fi madaallii bulchiinsaa, ETN Council, jalqabuuf karoorfameera.

### Waggaa 3 (2026-2027):

Waggaa sadaffaan bal'ina sirna ikkoo kanaa kan ETN Bio, ETN Council, giddugala guddaa ETN Home, fi madaallii walsimsiisaa ETN Join jalqabuun caalaatti bal'isuu irratti xiyyeeffata. Yeroon kunis, hojiiwwan fooyyessuu fi bal'isuu kanneen madaalliiwwan hojjechaa jiran hunda irratti hojjetaman dabalata.

Karaa misoomaa dhiyaate kunis, madaalliiwwan walitti hidhata qaban baay'ee kan yeroo waggaa sadii keessatti jalqabuu kan ibsu yoo ta'u, fedhii guddaa qabaachuun isaa hin shakkisiisu. Tooftaan sadarkaa sadarkaan kan madaallii tokko tokkoof fudhatame—kan karoora jalqabaa, sochii misoomaa addaa, qorannoo beta kan yaada fayyadamaa dabalatu, jalqaba ummataa ofiisaa, fi fooyya'iinsa itti aanu—tooftaa bulchiinsa piroojektii idilee fi sirrii dha. Haa ta'u malee, karoora misoomaa fi jalqabaa kan madaalliiwwan baay'ee kan yeroo tokkootti hojjetaman, qindoomina addaa, bulchiinsa qabeenyaa cimaa, fi garee dandeettii olaanaa qabu kan milkaa'inaan hojii irra oolchuuf barbaada. Milkaa'inni madaalliiwwan sadarkaa boodaa yeroo baay'ee dhiyeessii yeroo fi hojii irra oolmaa kanneen duraanii, kanneen bu'uuraa irratti hundaa'a (fkn, ETN Home, akka giddugala guddaatti, walitti dhufeenya fi qabiyyee hiika qabu kennuuf madaalliiwwan biroo hojii irra oolan barbaada). Jijjiirramni ifa ta'e kan karaa misoomaa sadarkaa baay'ee olaanaa kan waraqaa adii $ETN jalqabaa keessatti dhiyaate irraa gara karoora ji'a-ji'aan kan baay'ee bal'aa fi qulqullina qabu kan galmeewwan boodaa keessatti argamu, guddina piroojektii fi karoora tooftaa guddaa agarsiisa. Karaa misoomaa bal'aa kunis iftoomina barbaachisaa fi yeroo ifa ta'e kanneen qabeenya qabanif kennaatus, hojii irra oolmaan isaa cimaan rakkoo guddaa fi safartuu murteessaa kan dandeettii hojii fi qulqullina garee ETN ta'a.

## Milkaa'ina Gurguddoo Galmeeffaman

Sirni Ikkoo ETN duruu milkaa'ina gurguddoo adda addaa galmeesseera, kunis guddinaa fi misooma sochii qabu agarsiisa:

*   **Adoolessa 30, 2024:** Jalqaba Ultimate Netsa Bootcamp. Sochiin barnootaa kun, kutaa ETN Learn, barattoota kutaa 12ffaa Itoophiyaa 7,000 ol ta'an sagantaa leenjii cimaa ji'a lamaa keessatti milkaa'inaan galmeesseera. Barnoonni kunis Forex Trading, Blockchain Technology & Crypto, fi Seensa Web3 fi The Open Network dabalata. Sagantaan kunis, rakkoo barnootaa biyyaalessaa furuuf fi karaa barnootaa addaa kennuuf kan kaayyoo qabu waan ta'eef, baay'ee barbaachisaa ture.
*   **Fulbaana 7, 2024:** $ETN GekoTerminal irratti tarreeffame. Tokeniin $ETN madaallii hordoffii kiriptoo-saayinsii GekoTerminal irratti tarreeffamuu isaa milkaa'inaan galmeesseera, kunis mul'achuu isaa fi salphaatti argamuu isaa kan daldaltootaa fi investitootaaf fooyyesseera.
*   **Onkoloolessa 10, 2024:** Kennaa Hawaasaa kan Pavel Durovif. Mallattoo fakkeenya kan guyyaa dhalootaa Pavel Durov irratti, hawaasni ETN tokenoota $ETN hundeesaa Telegramif ergeera, kunis tokenoonni kunis guyyaa tokko gatii guddaa argachuu fi maallaqa bu'uuraalee serverii Telegramif gumaachuu danda'u jedhanii abdii waliigalaa ibsuun.
*   **Onkoloolessa 12, 2024:** ETN Ads Qorannoo Alfaa Cufame Seene. Madaallii ETN Ads, kan dhalootaan rakkoo hackathonii kan miseensa hawaasaan injifatame irraa dhufe, gara sadarkaa qorannoo alfaa cufameetti deemeera, kunis guddina haaraa kan oomisha hawaasaan geggeeffamu fi misooma agarsiisa.
*   **Sadaasa 17, 2024:** Sirna Eebbaa ETN Learn, Ahadu Badge SBT Minting, fi Jalqaba Laafaa Sirna Ikkoo ETN. Eebbi garee jalqabaa ("Ahadu Batch") kan Ultimate Netsa Summer Bootcamp irraa godhame, yeroo kana Ahadu Badge Soulbound Tokens (SBTs) yaadannoo kan blokcheenii TON irratti uumaman. Sagantaan kunis, jalqaba laafaa kan Sirna Ikkoo ETN bal'aaaf tajaajileera.
*   **Ebla 1, 2025:** Jalqaba Tapha Airdropii Telegram Kan Itoophiyaa Irratti Hundaa'e. Airdropii walqunnamtii qabu, kan tapha fakkaatu, Telegram irratti jalqabameera, kunis hirmaattotaaf ET Nesta coins (tilmaamaan $ETN) akka argatan dandeessiseera. Tapha kunis elementoota aadaa Itoophiyaa, kanneen akka "Jebena" (meeshaa bunaa kan durii) irraa fagaachuu, kan hawaasa kiriptoo addunyaa haala aadaa waliin walsimuun hirmaachisuuf dabalateera.

Milkaa'inniwwan kunis, keessattuu Ultimate Netsa Bootcamp fi tapha airdropii kan aadaa irratti hundaa'e, xiyyeeffannoo cimaa fi jalqabaa kan hirmaannaa hawaasaa bu'uuraa fi barnoota hojii irra ooluu irratti agarsiisu, kan haala Itoophiyaa irratti xiyyeeffannoo ifa ta'e qabu. Netsa Bootcamp, fakkeenyaaf, rakkoo barnootaa naannoo sanaa kallattiin furuun, yeroo tokkootti kumaatama dargaggoota Itoophiyaa Web3 fi blokcheenii TON waliin wal barsiiseera. Sochiin kunis, investii guddaa kan hawaasa ijaaruu fi dhiibbaa hawaasaa irratti agarsiisa. Haaluma walfakkaatuun, tapha airdropii Telegram tooftaa haaraa kan mallattoolee aadaa kanneen akka Jebena fayyadamuun fayyadamtoota hirmaachiseera, kunis muuxannoo sana caalaatti walitti dhufeenya qabu fi yaadatamoo taasiseera. Sochiilee kunis milkaa'ina piroojektii kiriptoo idilee, kan yeroo baay'ee tarreeffama tokenii yookiin odeeffannoo teeknikaa irratti qofa xiyyeeffatu, irra darbu. Kunis, kutannoo Sirna Ikkoo ETN kan ergaa hawaasaa isaa Itoophiyaa keessatti fi tooftaa isaa kan fayyadamaa hirmaannaa qabu fi barate horachuu agarsiisa. Kanaafuu, milkaa'inniwwan kunis, misooma hawaasaa fi barnoota hojii irra ooluu, kan aadaa waliin walsimu, akka bu'uura milkaa'ina yeroo dheeraa isaa kan misooma teeknikaa madaalliiwwan isaa tiif murteessaa ta'an agarsiisu.

## Garee Sirna Ikkoo ETN Geggeessu

Sirni Ikkoo ETN garee adda addaa kan miseensota isaa kanneen muuxannoo adda addaa kan teeknooloojii blokcheenii, misooma daldalaa, injinarii sooftiweerii, dizayinii, fi hirmaannaa hawaasaa qaban qabuun kan geggeeffamu dha. Odeeffannoon bal'aa waa'ee miseensota garee, hojii isaanii, duubee isaanii, fi gumaacha isaanii galmee sirna ikkoo bu'uuraa keessatti kennameera. Pitch deckis garee ogeessota muuxannoo qaban fi gorsitoota beekumsa qaban agarsiisa.

### Hogganummaa:

*   **Jason Peters:** Hundeesaa Sirna Ikkoo ETN. Akkasumas CEO ETHIO TECH Group LLC fi Hundeesaa ETHIO TECH AI yoo ta'u, hogganummaa mul'ata qabu kenna fi kallattii tooftaa sirna ikkoo kanaa geggeessa.

### Hundeesitoota Waliin fi Gorsitoota:

*   **Nathnael Biruk:** Hundeesaa Waliin ETN fi Hundeesaa CryptoTalk-ET. Mariiwwan kiriptoo-saayinsii fi hawaasa ijaaruu keessatti beekumsa isaa fayyadamuun guddina tooftaa ETN deeggara.
*   **Dawit Mengistu:** Gorsaa Sirna Ikkoo ETN fi Daayirektara Telegram Wallet Africa. Walsimsiisa waletii fi aplikeeshiniiwwan blokcheenii haala Afrikaa keessatti gorsa deeggarsa kenna.
*   **Nebiyu Sultan:** Gorsaa fi Dabalataa-Chain Developer kan Sirna Ikkoo ETN, kan Endubis Wallet waliin walqabatu. Bulchiinsa waletii fi tooftaalee faayinaansii waloo hin taane (DeFi) irratti gorsa kenna.

### Garee Misoomaa:

*   **Elias Taye:** Misoomaa Olaanaa Sirna Ikkoo ETN fi Misoomaa Bu'uuraa ETHIO TECH AI. Garee misoomaa geggeessa, AI walsimsiisuu fi furmaataalee blokcheenii irratti xiyyeeffata.
*   **Al Aymen:** Dizayinerii Weebii fi Misoomaa kan Sirna Ikkoo ETN, kan interface fayyadamaa dizayinii fi misooma, hojii salphaa fi muuxannoo fayyadamaa salphaa mirkaneessuuf itti gaafatama.

### Giraafiksii fi Quunnamtii:

*   **Biniam Samuel:** Dizayinerii Giraafiksii Olaanaa kan Sirna Ikkoo ETN fi Abbaa Qabeenyaa Waliin IDENTICO. Eenyummaa viizuwaliin ETN irratti hojjeta fi qabeenya brandii kan hirmaannaa qabu uuma.
*   **Asnake Gizaw:** Ofiseera Quunnamtii kan Sirna Ikkoo ETN fi miseensa Garee Gabaa Ethiopian Airlines. Quunnamtii fi hojiiwwan PR bulcha, walitti dhufeenya fi hirmaannaa ummataa jajjabeessa.

### Faayinaansii fi Hirmaannaa Hawaasaa:

*   **Mesay Feleke:** Qindeessaa Hawaasaa kan Sirna Ikkoo ETN fi moodeereetara hawaasa ET Netsa Apps. Sochiilee hawaasaa geggeessa, hirmaannaa fi fudhatama fayyadamaa jajjabeessa.
*   **Welid Sherif:** Qindeessaa Hawaasaa kan Sirna Ikkoo ETN, akkasumas moodeereetara hawaasa ET Netsa Apps, hojiiwwan kanneen sochiiwwan hawaasaa geggeessuu fi fudhatama fayyadamaa jajjabeessuu keessatti qooda fudhata.

### Miseensota Garee Waliigalaa:

*   **Molla Adera:** Miseensa garee kan deeggarsa adda addaa kennuun guddina fi milkaa'ina sirna ikkoo ETN waliigalaa irratti gumaacha.

Walitti qabamni garee Sirna Ikkoo ETN walitti makamuu tooftaa kan beekumsa Itoophiyaa naannoo fi muuxannoo addunyaa irratti hundaa'e agarsiisa, kan xiyyeeffannoo guddaa kan walitti dhufeenya naannoo fi hubannoo qabu. Miseensonni garee hedduun walitti dhufeenya ifa ta'e kan sochiiwwan Itoophiyaa, daldaltoota, yookiin hojiiwwan kanneen gabaa Afrikaa waliin kallattiin walqabatan qabu, kanneen akka ETHIO TECH, CryptoTalk-ET, Endubis Wallet, fi Ethiopian Airlines. Dabalatni gorsitoota kanneen akka Dawit Mengistu, kan Daayirektara Telegram Wallet Africa ta'an, beekumsa hojii murteessaa kan Afrikaa qaba. Hogganummaan Jason Peters, hojii isaa kan CEO ETHIO TECH Group keessatti, ijaarsa dhaabbataa fi deeggarsa sirna ikkoo kanaaf agarsiisa. Walitti makamni beekumsa naannoo fi beekumsa teeknikaa fi tooftaa bal'aa kunis, piroojektii kan furmaataalee Web3 kanneen fedhii fi haala Afrikaaf qophaa'an misoomaa fi hojii irra oolchuuf kaayyoo qabuuf murteessaa dha. Kanaafuu, ijaarsi garee kunis rakkoolee teeknikaa walxaxaa kanneen misooma blokcheenii keessatti argaman fi tooftaa gabaa addaa fi aadaa naannoo isaanii kanneen bu'uuraa ta'an keessa darbuuf qophaa'eera.
