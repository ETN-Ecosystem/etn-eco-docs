# Seensa Sirna Ikkoo ETN

Sirni Ikkoo ETN jalqaba teeknooloojii blokcheenii gurguddaa ta'ee, On The Open Network (TON) blokcheenii irratti haala qulqullina qabuun ijaarame. Kaayyoon isaa inni guddaan, diigitaalii fi aplikeeshiniiwwan waloo hin taaneef hojiiwwan dhugaa, addunyaa irratti mul'atan uumuu dha, keessattuu Afrikaa keessatti namootaa fi daldaltoota humneessuu irratti xiyyeeffata. Mul'anni isaa inni olaanaan, sirna faayinaansii waloo hin taane fi hunda hammate hundeessuu dha. Mul'anni kunis, teeknooloojii blokcheenii fayyadamuun tajaajiloota faayinaansii salphaatti argaman, iftoomina qaban, fi nageenya qaban kennuun hojii irra oola.

## Mul'ata, Ergaa, fi Falaasama "Netsa"

Onnee eenyummaa Sirna Ikkoo ETN irratti jechi Amaariffaa "Netsa" (ነፃ) jedhu jira, kunis "bilisa," "birmadummaa," yookiin "walabummaa" jechuun hiikama. Jechi kun maqaa qofa osoo hin taane, bu'uuraalee sirnichaa kanneen geggeessan kan of keessaa qabu dha. Kunis, bu'uuraalee waloo hin taane fi humneessuu kanneen teeknooloojii blokcheenii TON irratti hojii irra oolan irratti xiyyeeffannoo guddaa qabaachuu agarsiisa. Filannoon "Netsa" jedhu, fayyadamtoota isaa bilisummaa faayinaansii horachiisuuf kutannoo cimaa qabaachuu isaa agarsiisa. Hidda bu'uura aadaa qabaachuun kun filannoo tooftaa qophaa'e dha. Jecha naannoo tokkoo bu'uura tokkoof fayyadamuun, piroojektiin kun gufuuwwan hubannoo hir'isuu fi abbummaa dhalootaa fi walitti dhufeenya uumuu irratti xiyyeeffata. Haalli kun piroojektoota kiriptoo addunyaa irratti maqaa waliigalaa yookiin teeknikaa qofa fayyadaman irraa adda. Kanaafuu, "Netsa" yaada Web3 walxaxaa ta'e, hawaasa Itoophiyaa fi Afrikaa bal'aa keessatti salphaatti hubatamaa fi walitti dhufeenya qabaataa akka ta'u taasisuuf riqicha yaadaa ta'ee tajaajila. Bu'uuraalee aadaa beekamoo irratti hundaa'uun kun, kaayyoo piroojektichaa salphaatti hubatamaa fi fedhii naannoo waliin kan walsimu taasisuun fudhatama saffisiisa jedhamee eegama. Tooftaan kunis, piroojektoota Web3 naannoo biroof fakkeenya ta'uu danda'a, akkamitti walsimsiisni aadaa hirmaannaa hawaasaa fi hojii piroojektichaa guddisuu akka danda'u agarsiisa.

Ergaan Sirna Ikkoo ETN baay'ee walxaxaa dha. Kunis, hawaasa ETN token irratti xiyyeeffate, kan sochii qabu fi hirmaannaa qabu horachuu dabalata. Tokkoon tokkoon ergaa kanaa inni guddaan, ETN gara ET Netsa Apps™ suiteitti walsimsiisuu dha, kunis muuxannoo fayyadamaa salphaa fi hirmaannaa qabu uumuu dha. Dabalataanis, piroojektiin kun sagantaalee barnootaa bal'aa, hirmaannaa hawaasaa cimaa, fi sochiiwwan guddinaa itti fufiinsaa qaban fayyadamuun tajaajiloota isaa fi token isaa bal'inaan akka fudhatamu gochuuf kutannoo qaba.

## Sirna Ikkoo ETN Blokcheenii TON Irratti: Ilaalcha Waliigalaa

Sirni ikkoo kunis, humna blokcheenii TON – saffisa, nageenya, fi bal'ina isaa – fayyadamuun tajaajiloota adda addaa walitti fida. Kunis, sirna maqaa doomeenii (DNS), tajaajila weebii, sagantaalee barnootaa, fi furmaataalee kaffaltii dabalata, kanneen hundi sirna waloo fi waloo hin taane keessatti hojjetan. Kaayyoon isaa inni guddaan, fayyadamtootaaf muuxannoo hojii irra ooluu fi bu'aa qabu, addunyaa diigitaalii fi qabeenya isaa keessatti kennuu dha. Hojiin piroojektichaa inni ofiisaa GitHub irratti argamu, ergaa kana caalaatti ibsa, kunis teeknooloojii blokcheenii fi hojiiwwan dhugaa addunyaa walitti fiduu fi dinagdee guddachaa jiru keessatti, ETN token meeshaa bu'uuraa ta'ee, haaraa uumuu irratti xiyyeeffata.

Filannoon blokcheenii TON bu'uura tooftaa sirna ikkoo kanaa ti, keessattuu xiyyeeffannoo Afrikaa qabaachuun isaa. Murtoon kunis, fedhii malee kan hin taane, garuu rakkoolee hawaas-dinagdee fi bu'uuraalee misoomaa naannoo sana keessatti mul'atan furuuf kan walsimu dha. Kaffaltiin blokcheenii TON xiqqaa ta'uu fi saffisni isaa guddaa ta'uun, fayyadamtoota Afrikaa hedduu kanneen furmaataalee blokcheenii akka fedhii qofa osoo hin taane, akka barbaachisummaa tokkootti ilaalan, sababa tajaajila faayinaansii idilee argachuun isaanitti ulfaatuuf, galchuuf murteessaa dha. Fudhatamni bal'aan aplikeeshinii ergaa Telegram Afrikaa keessatti argamu, fayyadamtootaaf gara sirna ikkoo TONitti karaa salphaa fi beekamaa ta'e kenna, kunis walsimsiisa waloo hin taane kan boorsaa TON ofiisaa waliin kan walitti dhufeenya qabuun caalaatti salphifama, kunis gufuuwwan yeroo aplikeeshinii boorsaa haaraa fudhatan mul'atan hir'isa. Dabalataanis, bal'inni TON inni dhalootaa, Sirni Ikkoo ETN guddina interneetii fi fudhatama diigitaalii Afrikaa keessatti eegamu, osoo hojii isaa hin hir'isin yookiin baasii isaa hin dabalin, akka fudhatu mirkaneessa. Kanaafuu, TON bu'uura teeknikaa qofa osoo hin taane; humneessaa tooftaa qophaa'e dha, kan kaayyoo naannoo sirna ikkoo kanaa, kanneen bu'uuraalee murteessoo ta'an kanneen akka gatii salphaa, salphaatti argamuu, fi bal'ina, kanneen dinagdee guddachaa jiru keessatti baay'ee barbaachisoo ta'an furuun, kallattiin deeggaru dha. Humnoonni blokcheenii TON, kanaafuu, dandeettii Sirna Ikkoo ETN ergaa isaa kan dabalata faayinaansii fi humneessuu Afrikaa keessatti hojii irra oolchuu isaa waliin walitti dhufeenya qabu, kunis tajaajiloota isaa hojii irra ooluu fi bal'inaan argamuu danda'an taasisa.
