# Walitti Qabama NFT Warra Eegduu Itoophiyaa

Warri Eegduu Itoophiyaa walitti qabama NFT addaa kan aadaa, seenaa, fi dhaalmaya Itoophiyaa artii diigitaalii fayyadamuun kabajuu dha.

## Ilaalcha Waliigalaa Walitti Qabamaa

### Yaada
- Bakka bu'iinsa diigitaalii kan elementoota aadaa Itoophiyaa
- Namoota seenaa fi iddoowwan beekamoo
- Tooftaalee artii aadaa
- Hiikaa ammayyaa

### Odeeffannoo Walitti Qabamaa
- Qabeenya Waliigalaa: NFT 10,000
- Blokcheenii: TON
- Foormaat: Artii Diigitaalii
- Sadarkaa Dhabamummaa: Waliigalaa, Dhabamaa, Epic, Lejendarii
