# Bu'uuraalee Teeknikaa fi Walsimsiisa Blokcheenii TON

Ijaarsi teeknikaa Sirna Ikkoo ETN Blokcheenii The Open Network (TON) waliin kan walitti hidhata cimaa qabu yoo ta'u, hojiiwwan isaa addaa fayyadamuun tajaajiloota waloo hin taane kanneen ummata isaa kan xiyyeeffateef, keessattuu Afrikaa keessatti, ijaara.

## TON Fayyadamuu: Faayidaalee Sirna Ikkoo ETN Afrikaa Keessatti

Filannoon TON akka blokcheenii bu'uuraa Sirna Ikkoo ETN tooftaa qophaa'e yoo ta'u, kunis faayidaalee adda addaa kanneen haala Afrikaaf baay'ee barbaachisoo ta'an irraa kan ka'e dha. Ijaarsi TON kaffaltii transakshinii gadi aanaa fi saffisa transakshinii ariifataa kan of keessaa qabu yoo ta'u, kunis fayyadamtoota Afrikaa gidduutti fudhatama jajjabeessuuf qabxiiwwan murteessoo dha. Naannoo kana keessatti namoota hedduuf, sirna kaffaltii onlaayinii idilee fi bu'uuraalee faayinaansii cimaa argachuun daangeffamee waan jiruuf, tooftaalee blokcheenii irratti hundaa'an kanneen gatii salphaa fi hojii irra ooluu danda'an mijataa qofa osoo hin taane, barbaachisummaa dha.

Dabalataanis, fudhatamni bal'aan aplikeeshinii ergaa Telegram Afrikaa guutuu keessatti argamu, bu'uura fayyadamaa uumamaa fi bal'aa kan tajaajiloota TON irratti hundaa'anitti salphaatti galuu danda'u kenna. Walsimsiisni waletii TON ofiisaa kan Telegram keessatti argamu, fayyadamtoota haaraaf gufuu galmee salphisa, kunis waletiiwwan adda addaa, kanneen hin beekamne, buufachuu, fe'achuu, fi akkamitti fayyadamuu akka qaban barachuu dhabamsiisa. Bal'inni TON kan mirkanaa'e faayidaa guddaa biraa yoo ta'u, kunis Sirni Ikkoo ETN guddina fuulduraa kan interneetii fi beekumsa diigitaalii kanneen qaaricha guutuu keessatti bal'achaa jiran, osoo rakkoo hojii yookiin dabala baasii guddaa hin qabaatin, akka fudhatu mirkaneessa.

Dinagdeen diigitaalii Afrikaa guddina guddaa agarsiisaa jira, kan gabaawwan e-commerce fi e-payments bal'ina guddaa argachuuf karoorfaman. Yeroo tokkootti, jijjiirramni dinagdee addunyaa kan dhiibbaa Global South irraa dabalaa jiru, kan Afrikaa dabalatu, ni mul'ata. Sirni Ikkoo ETN, humna TON irratti ijaaruun fi fedhiiwwan naannoo kanneen akka furmaataalee kaffaltii salphaatti argaman fi madaalliiwwan e-commerce kallattiin furuun, jijjiirama diigitaalii kana fayyadamuuf tooftaa qophaa'een argama. Kaayyoon isaa, dinagdee diigitaalii Afrikaa kan guddachaa jiru keessatti hirmaachuu qofa osoo hin taane, hojii irra oolchuufis. Walitti makamni baay'ina fayyadamtoota interneetii Afrikaa kan guddachaa jiru fi gabaawwan diigitaalii kanneen bal'achaa jiran, akkasumas rakkoolee itti fufiinsaa kanneen tajaajila faayinaansii idilee argachuu waliin walqabatan, iddoo gaarii kan sochiiwwan akka ETN tiif uuma. Hojiilee TON addaa—kaffaltii gadi aanaa, saffisa guddaa, fi walsimsiisa Telegram dhalootaa—rakkoolee jiran kana furuuf baay'ee mijatoo dha. Sagantaan madaalliiwwan kanneen ETNn kennaman, kan ETN Gebeya, ETN Pay, ETN Equb, fi ETN Learn dabalatu, meeshaalee barbaachisoo kan namootaa fi daldaltoota Afrikaa Web3 dinagdee keessatti kallattiin hirmaachuuf kenna. Jijjiirramni bal'aan kan "Global South" dinagdeedhaan beekamtii argachuu isaa, yeroo fi dhiibbaa sochii kanaa caalaatti agarsiisa. Kanaafuu, ETN tooftaa qophaa'een akka humneessaa bu'uuraa kan fayyadamtootaa fi daldaltoota Afrikaa sirna durii irra darbuun fi dinagdee addunyaa waloo hin taane keessatti guutummaatti fi haqa qabeessa ta'een hirmaachuuf argama.

## Teeknooloojiiwwan Bu'uuraa TON Kan Fayyadan

Madaalliiwwan Sirna Ikkoo ETN tokenii TON irratti jalqabuu qofa osoo hin taane, hojiiwwan isaa adda addaa ijaaruuf dandeettiiwwan Layer-1 dhalootaa TON bal'aa kanneen adda addaa walitti fidu. Fayyadamni tooftaa qophaa'e kan tooftaa TON kanaa, kan hojii irra oolmaa tokenii salphaa irra darbu, ETN tajaajiloota dhugaa waloo hin taane kanneen nageenya, hojii, fi hojiiwwan addaa TON irraa kallattiin fayyadan kennuuf dandeessisa. Kunis, Sirna Ikkoo ETN agarsiisa cimaa kan dandeettii TON akka blokcheenii L1 bal'aa kan sirna ikkoo dApp walxaxaa fi baay'ee walxaxaa ta'e ijaaruuf mijataa ta'e taasisa. Teeknooloojiiwwan TON addaa kanneen fayyadan kanneen keessaa:

*   **TON Connect SDK:** Meeshaan Misoomaa Sooftiweerii kunis walsimsiisa waletii nageenya qabu fi salphaa kan madaalliiwwan ETN baay'ee keessatti argamuuf murteessaa dha. ETN Pay (saayinii transakshinii tiif), ETN Council (abbummaa NFT mirkaneessuuf sagalee bulchiinsaa tiif), fi ETN Join (walitti dhufeenya waletii kanneen walitti dhufeenya qabanif salphisuuf) keessatti fayyadamuu isaa ifatti ibsameera. Odeeffannoon waliigalaa waa'ee TON Connect hojii isaa akka sadarkaa hayyama aplikeeshinii addunyaa kan sirna ikkoo TON keessatti mirkaneessa.
*   **TON Storage:** Furmaanni kuusaa waloo hin taane kunis kan blokcheenii TONn kennamu bu'uura ETN Hosting ti, kan qabiyyee weebsaayitii kuufamu.
*   **TON DNS:** Madaallii ETN DNS, kan fayyadamtootaaf maqaa doomeenii gadii akka NFTtti bitachuu fi uumuu dandeessisu, tajaajila TON DNS dhalootaa irratti ijaarame.
*   **TON Multisig Wallets:** Waletiiwwan baay'ee-saayinii kunis bulchiinsa maallaqaa nageenya qabu kan madaallii ETN Equb keessatti fayyadu, akkasumas kaffaltii madaallii sirna ikkoo guutuu keessatti walitti qabuuf kan ramadaman yoo ta'u, kunis raabsa waggaa kanneen qabeenya qabanif dura.
*   **TON NFTs (Jettons & sadarkaa NFT waliigalaa):** Non-Fungible Tokens hojii bu'uuraa kan tajaajiloota ETN adda addaa keessatti argamu. Maqaa doomeenii gadii ETN DNS akka NFTtti uumamu. Walitti qabamni "Warra Eegduu Itoophiyaa" NFTwwan qooda bu'aa qaban of keessaa qaba. ETN Numbers kan haaraa ibsame walitti qabama NFT ti. Dabalataanis, Soulbound Tokens (SBTs) ETN Learn keessatti ragaa kennuuf fayyadamuu isaa agarsiisa. Blokcheenii TON sadarkaa ifa ta'e kan tokenoota fungible (Jettons, akka $ETN ofiisaa) fi akaakuuwwan NFT adda addaaf qaba.
*   **TonAPI:** Walitti qabama APIwwan HTTP sadarkaa olaanaa kan plugin WooCommerce ETN Pay fayyadu kan hojiiwwan murteessoo kanneen akka gatii $ETN yeroo dhugaa argachuu (karaa /v2/rates endpoint) fi seenaa transakshinii argachuun transakshinii mirkaneessuuf.

## Ilaalcha Waliigalaa Fayyadama Smart Contractii Sirna Ikkoo Keessatti

Madaalliiwwan fi tajaajiloota adda addaa kanneen Sirna Ikkoo ETN keessatti argaman, misooma smart contractii bal'aa fi adda addaa barbaadu, kan loojikii on-chain kan hojii isaa bu'uuraa ta'u. Kunis, investii guddaa kan smart contractiiwwan adda addaa, kanneen sadarkaa tokenii, hojiiwwan NFT, tooftaalee dhaabbata waloo hin taane kan of danda'e (DAO), fi loojikii faayinaansii kan DeFi fakkaatu kan madaalliiwwan akka ETN Equb tiif qophaa'an dabalatu.

*   **Contractii Tokenii $ETN:** Tokeniin $ETN bu'uuraa Jetton (sadarkaa tokenii fungible kan TON) smart contractii dha. Teessoon contractii isaa guddaa EQAz_XrD0hA4cqlprWkpS7TIAhCG4CknAfob1VQm-2mBf5Vl jedhamee ibsameera. Waraqaan Adii $ETN jalqabaa kan contractii kanaa kan gatii hir'isuu danda'u, kan afaan sagantaa Tact keessatti barreeffame, irratti mari'ateera, garuu tooftaan gatii hir'isuu kunis booda tooftaa raabsa kaffaltii filachuun didameera.
*   **ETN Equb:** Madaalliiwwan kunis smart contractiiwwan irratti hundaa'uun hojiiwwan bu'uuraa sirna Equb kan durii, kanneen akka gumaacha miseensaa bulchuu, sagantaa kaffaltii naanna'aa, fi raabsa, ofumaan hojjetu.
*   **ETN DNS:** Maqaa doomeenii gadii kan ETN DNSn argaman akka NFTtti uumamu. Hojiin kunis, hojiiwwan smart contractii kanneen galmee doomeenii, abbummaa, jijjiirraa, fi tooftaalee bitannaa/gurgurtaa bulchan of keessaa qaba.
*   **ETN Numbers & ETN Gulit:** Akka walitti qabama NFTtti, ETN Numbers NFTwwan "Lakkoofsa" dhuunfaa uumuuf fi metadata walitti qabamaa bulchuuf smart contractiiwwan barbaada. Gabaan NFT ETN Gulit kan walqabatu, walitti qabama smart contractiiwwan walxaxaa kanneen tarreeffama, bitannaa, gurgurtaa, fi tarii raabsa rooyaalitii bulchan irratti hundaa'a. Sadarkaa NFT TON waliigalaa fi tooftaalee gabaa kanaaf bu'uura kenna.
*   **Warra Eegduu Itoophiyaa NFTs:** Walitti qabama NFT kunis, kan tooftaa qooda bu'aa addaa kan madaallii sirna ikkoo addaa waliin walqabatu, smart contractiiwwan dabalataanis ni bulfama. Contractiiwwan kunis uumuu NFT, abbummaa, fi tarii loojikii kan qooda bu'aa kanneen NFT qabanif herreguu fi raabsuu bulchu.
*   **ETN Council (DAO):** Madaallii bulchiinsaa smart contractiiwwan fayyadamuun yaada dhiyaate galmeessuuf fi sagalee lakkaa'uuf. Abbummaan NFTwwan Bulchiinsaa fi Treasury Bonds, kanneen humna sagalee kennuu murteessan, on-chain irratti mirkanaa'u, kunis walqunnamtii smart contractii kanneen meeshaalee bulchiinsaa kanaaf dabalata.
*   **Soulbound Tokens (SBTs):** Maqaa Ahad-Badge-minter, kan jayp89/ton-sbt-minter irraa kan fudhatame fi ETN Learn waliin walqabatu, blokcheenii TON irratti SBTwwan uumuuf smart contractiiwwan fayyadamuu isaa agarsiisa, tarii ragaa yookiin milkaa'ina hin jijjiiramne kennuuf.

### Afaanota Misoomaa:

Sirni ikkoo kunis Tact fayyadamuun smart contractiiwwan barreessa, akka fakkeenya $ETN Whitepaper keessatti argamu fi GitHub irratti ibsame. Ahad-Badge-minter TypeScript keessatti kan jiru yoo ta'u, tarii off-chain scripting kan smart contractiiwwan TON waliin walqunnamuuf.

Bal'inni fi walxaxinni madaalliiwwan kanaa Sirna Ikkoo ETN walitti qabama weeb-front-endii qofa osoo hin taane, walitti qabama tajaajiloota on-chain kanneen walitti hidhata cimaa qaban ta'uu isaa agarsiisa. Kunis, bu'uuraalee smart contractii cimaa fi fooyya'aa barbaada, kan dandeettii misooma teeknikaa guddaa kan garee keessatti yookiin gumaachitoota hawaasaa irraa argamu agarsiisa. Fayyadamni Tact fi muuxannoon kan SBT minters waliin agarsiifame, hojii misoomaa sochii qabu fi dandeettii qabu kan blokcheenii TON irratti hojjetamu caalaatti agarsiisa.
