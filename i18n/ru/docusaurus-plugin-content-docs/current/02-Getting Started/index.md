# Добро пожаловать в Экосистему ETN

**Эфиопия без границ**

Экосистема ETN — это набор децентрализованных протоколов, платформ и приложений, построенных на **блокчейне TON**. Наша миссия — расширить возможности Эфиопии и африканского континента с помощью цифровой идентичности, финансовой доступности и устойчивых к цензуре технологий.

## С чего начать

*   **[Введение в платформу](../04-platforms/index.md)**: Узнайте о различных компонентах экосистемы, таких как ETN Pay, ETN Learn и других.
*   **[Токеномика $ETN](../05-etn-token/01-tokenomics.md)**: Поймите роль токена $ETN как топлива нашей цифровой экономики.
*   **[ETN Identity](../04-platforms/etn-identity/index.md)**: Исследуйте нашу суверенную систему идентификации.

## Для разработчиков

*   **[ETN Identity SDK](../04-platforms/etn-identity/etn-identity-sdk.md)**: Интегрируйте "Войти через ETN" в свои приложения.

Присоединяйтесь к нам в построении цифрового будущего Африки.
