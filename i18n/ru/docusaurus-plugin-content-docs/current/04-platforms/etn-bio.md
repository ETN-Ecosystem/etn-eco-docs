# ETN Bio

## Платформа листинга био-ссылок – В ожидании

**ETN Bio** — это платформа списков ссылок для биографии (bio link), похожая на Linktree, с дополнительным бонусом в виде возможности интеграции кошелька и приема чаевых в ETN. Она предоставляет централизованный профиль для пользователей, чтобы продемонстрировать все свое онлайн-присутствие.
