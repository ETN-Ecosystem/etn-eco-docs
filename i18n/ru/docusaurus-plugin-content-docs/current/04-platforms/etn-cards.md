# ETN CARDS

## Платформа выпуска виртуальных карт – В ожидании

**ETN Cards** — это платформа для выпуска виртуальных карт. Первоначальная покупка этих виртуальных карт будет осуществляться в ETN, но затем пользователи смогут пополнять карты с помощью USDT для совершения онлайн-покупок, связывая экосистему ETN с более широкой цифровой коммерцией.
