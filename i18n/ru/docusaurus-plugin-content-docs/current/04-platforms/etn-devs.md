# ETN Devs

## Платформа фриланса для разработчиков – Работает

**ETN Devs** — это маркетплейс фриланса специально для разработчиков и выполнения эпизодических работ. Он связывает разработчиков с проектами внутри и за пределами экосистемы ETN, используя смарт-контракты для безопасного и справедливого выпуска оплаты, подобно ETN Netsa-Lancer.
