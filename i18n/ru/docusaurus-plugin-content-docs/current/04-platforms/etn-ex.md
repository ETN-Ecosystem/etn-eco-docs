# ETN-Ex

## Агрегатор децентрализованных бирж – В ожидании

**ETN-Ex** задуман как агрегатор децентрализованных бирж на блокчейне TON. Его цель — предоставить пользователям комплексную платформу для доступа и использования различных сервисов децентрализованного обмена в экосистеме TON, облегчая торговлю и ликвидность для ETN и других токенов.
