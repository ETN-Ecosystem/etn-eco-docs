# ETN HOSTING

## Сервисы хостинга и обратного прокси на базе TON – Закрытая Бета

**ETN Hosting** предоставляет услуги децентрализованного веб-хостинга, используя TON Storage, что позволяет пользователям размещать свои веб-сайты непосредственно на блокчейне TON. Он также предлагает услуги обратного прокси для интеграции сайтов Web2 с TON. Все планы хостинга и сопутствующие услуги оплачиваются токенами ETN.
