# ETN Jem’aa

## Наша собственная платформа социальных сетей Web 3.0 – В ожидании

**ETN Jem'aa** — это собственная платформа социальных сетей экосистемы на базе Web 3.0. Она направлена на создание децентрализованного пространства для взаимодействия сообщества, обмена контентом и вовлечения, используя технологию блокчейн для улучшения контроля пользователей и конфиденциальности.
