# ETN Kazna

## Физический холодный кошелек – Работает

**ETN Kazna** — это физические аппаратные кошельки (холодные кошельки) для ETN, произведенные в Эфиопии. Эти кошельки обеспечивают безопасный, оффлайн метод хранения токенов ETN, повышая варианты безопасности, доступные пользователям в экосистеме.
