# ETN Numbers

## NFT коллекция – В ожидании

**ETN Numbers** — это NFT-коллекция, похожая на анонимные номера Telegram. Эти NFT позволят пользователям входить в экосистему ETN анонимно или использовать NFT в качестве двухфакторной аутентификации или генератора паролей при входе в систему, являясь частью инициативы единого входа (SSO).
