# Глоссарий ключевых терминов

Этот глоссарий содержит определения ключевых терминов, платформ и технологий, относящихся к Экосистеме ETN.

*   **$ETN (ET Netsa Coin):** Нативная утилитарная криптовалюта Экосистемы ETN, построенная на блокчейне TON, используемая для транзакций, комиссий, стейкинга, управления и вознаграждений на всех платформах экосистемы.
*   **ETN Ads (ads.ETnetsaCoin.ton):** Децентрализованный рекламный портал в Экосистеме ETN.
*   **ETN Auth:** Платформа аутентификации в наборе ETN Numbers.
*   **ETN Authenticator:** Инструмент или приложение, предназначенное для облегчения процесса ETN Auth.
*   **ETN Bio (bio.ETnetsaCoin.ton):** Сервис децентрализованных био-ссылок, размещенный на блокчейне TON.
*   **ETN Council (dao.ETnetsaCoin.ton):** Платформа управления экосистемой ETN.
*   **ETN DNS (ETN Domain Name System):** Система доменных имен на основе блокчейна.
*   **ETN Equb:** Блокчейн-адаптация традиционной эфиопской ротационной сберегательно-кредитной ассоциации (ROSCA).
*   **ETN Gebeya (ранее ETN Sell - sell.ETnetsaCoin.ton):** Платформа электронной коммерции.
*   **ETN Guilt:** NFT-маркетплейс в наборе ETN Numbers.
*   **ETN Home (home.ETnetsaCoin.ton):** Официальный веб-сайт и центральный хаб экосистемы ETN.
*   **ETN Hosting:** Услуги децентрализованного веб-хостинга.
*   **ETN Join (join.ETnetsaCoin.ton):** Платформа заявок для крупных проектов, стремящихся интегрироваться.
*   **ETN Learn (learn.ETnetsaCoin.ton):** Платформа онлайн-обучения, ориентированная на Web3.
*   **ETN Numbers:** NFT-коллекция на блокчейне TON.
*   **ETN Pay:** Безопасное решение для обработки платежей.
*   **Amharic (Амхарский):** Семитский язык, на котором говорят в Эфиопии.
*   **DeFi (Decentralized Finance):** Финансовая система, построенная на технологии блокчейн.
*   **GekoTerminal:** Платформа для отслеживания криптовалют.
*   **Guardians of Ethiopia:** NFT-коллекция из 20 эфиопских супергероев.
*   **Jetton (Жетон):** Стандарт для взаимозаменяемых токенов на блокчейне TON.
*   **NFT (Non-Fungible Token):** Уникальный цифровой актив.
*   **Netsa (ነፃ):** Амхарское слово, означающее «свободный», «свобода» или «освобождение».
*   **ROSCA:** Традиционная неформальная финансовая система (Equb).
*   **SBT (Soulbound Token):** Тип непередаваемого NFT.
*   **Tact:** Язык программирования для написания смарт-контрактов на TON.
*   **Telegram:** Популярная платформа обмена сообщениями.
*   **TON Blockchain (The Open Network):** Быстрый, безопасный и масштабируемый блокчейн уровня 1.
*   **TON Connect SDK:** Комплект разработки ПО для подключения кошельков.
*   **TON DNS:** Децентрализованная система доменных имен.
*   **TON Multisig Wallet:** Тип криптовалютного кошелька, требующий нескольких подписей.
*   **TON Storage:** Децентрализованная система хранения файлов.
*   **TonAPI:** Набор высокоуровневых HTTP API.
*   **Web3:** Концепция следующего поколения интернета.
*   **Ahadu Badge:** Памятный Soulbound Token (SBT), присуждаемый первым выпускникам ETN Learn.
*   **$ETN Treasury Bonds:** Актив управления в рамках Совета ETN.
*   **ETN Authenticator:** Автономная система двухфакторной аутентификации (2FA).
*   **ETN Numbers NFTs:** Специфические NFT, используемые ETN Authenticator.
*   **Mesewer Books:** Недавно анонсированная платформа в Экосистеме ETN.
*   **NFT-Based Governance:** Модель принятия решений Экосистемы ETN, где права голоса привязаны к владению NFT.
