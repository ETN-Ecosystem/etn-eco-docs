# Технические основы и Синергия с блокчейном TON

Техническая архитектура Экосистемы ETN глубоко переплетена с блокчейном The Open Network (TON), используя его специфические особенности для построения набора децентрализованных сервисов, адаптированных для целевой аудитории, особенно в Африке.

## Использование TON: Преимущества для Экосистемы ETN в Африке

Выбор TON в качестве фундаментального блокчейна для Экосистемы ETN является стратегическим, обусловленным несколькими преимуществами, особенно актуальными для африканского контекста. Архитектура TON предлагает низкие комиссии за транзакции и высокую скорость, что является критическими факторами для стимулирования принятия пользователями в Африке.

Кроме того, широкое распространение мессенджера Telegram по всей Африке обеспечивает естественную и обширную базу пользователей, которых можно легко познакомить с сервисами на базе TON. Бесшовная интеграция официального кошелька TON внутри Telegram значительно снижает барьер для входа новых пользователей.

Экосистема ETN, опираясь на сильные стороны TON и напрямую решая местные проблемы, такие как доступные платежные решения и платформы электронной коммерции, стратегически позиционирована, чтобы извлечь выгоду из цифровой трансформации Африки.

## Используемые технологии ядра TON

Платформы Экосистемы ETN не просто запускают токен на TON, но глубоко интегрируют широкий спектр нативных возможностей Layer-1 TON:

*   **TON Connect SDK:** Этот комплект разработки программного обеспечения (SDK) имеет решающее значение для обеспечения безопасной и бесшовной интеграции кошельков на нескольких платформах ETN (ETN Pay, ETN Council, ETN Join).
*   **TON Storage:** Это децентрализованное решение для хранения данных является основой для ETN Hosting.
*   **TON DNS:** Платформа ETN DNS, позволяющая пользователям делать ставки и минтить поддомены как NFT, построена на нативном сервисе TON DNS.
*   **TON Multisig Wallets:** Мультисиг-кошельки используются для безопасного управления средствами в рамках платформы ETN Equb и для накопления комиссий платформы.
*   **TON NFTs (Jettons & общий стандарт NFT):** NFT являются ключевым элементом в нескольких сервисах ETN (ETN DNS, Guardians of Ethiopia, ETN Numbers).
*   **TonAPI:** Этот набор высокоуровневых HTTP API используется плагином ETN Pay для WooCommerce для критических функций, таких как получение цен $ETN в реальном времени.

## Обзор использования смарт-контрактов в экосистеме

Разнообразный спектр платформ и сервисов в Экосистеме ETN требует обширной разработки смарт-контрактов:

*   **Контракт Токена $ETN:** Основной токен $ETN сам по себе является смарт-контрактом Jetton (стандарт взаимозаменяемых токенов TON).
*   **ETN Equb:** Эта платформа полагается на смарт-контракты для автоматизации основных функций традиционной системы Equb.
*   **ETN DNS:** Поддомены, приобретенные через ETN DNS, минтятся как NFT.
*   **ETN Numbers и ETN Guilt:** Как NFT-коллекция, ETN Numbers потребует смарт-контрактов для минтинга и управления метаданными.
*   **Guardians of Ethiopia NFTs:** Эта NFT-коллекция с механизмом распределения прибыли также будет управляться смарт-контрактами.
*   **ETN Council (DAO):** Платформа управления использует смарт-контракты для записи предложений и подсчета голосов.
*   **Soulbound Tokens (SBTs):** Связанные с ETN Learn для выдачи непередаваемых удостоверений.

### Языки разработки:

Экосистема использует **Tact** для написания смарт-контрактов, о чем свидетельствуют примеры в Белой книге $ETN и упоминания на GitHub.
